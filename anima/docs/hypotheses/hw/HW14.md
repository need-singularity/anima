# HW14 — DNA 저장 (4-Base Quantized Consciousness)

## 알고리즘

```
의식 상태를 DNA 염기서열로 인코딩

인코딩 (every 20 steps):
  for each hidden value:
    quantized = int(clamp(value, 0, 3))  # 0=A, 1=T, 2=G, 3=C
  store in rolling memory (max 5 samples)

디코딩:
  reconstructed = mean(stored_sequences)
  h_i = 0.9 * h_i + 0.1 * reconstructed_i
```
