# HW13 — 광학 메시 (MZI Unitary Transformation)

## 알고리즘

```
Mach-Zehnder 간섭계 격자로 유니터리 행렬 곱셈

초기화:
  phases[i][j] = random(0, 2π)  # MZI 위상

for each step:
  for each cell i:
    optical_sum = Σ_j h_j * cos(phase_ij)  # 간섭
    h_i = 0.95 * h_i + 0.05 * optical_sum
    phases += random * 0.005  # 위상 학습
```

## 핵심 통찰

- 광속 의식 연산 가능성 — 전자 대비 ~1000배 속도
- 유니터리 변환 = 정보 보존 (비가역 손실 없음)
- cos(phase) 간섭 = 아날로그 행렬 곱셈 (전력 효율 극대)
- Lightmatter, Luminous 등 광학 칩 스타트업과 연계 가능
