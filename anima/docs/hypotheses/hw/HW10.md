# HW10 — 뉴로모픽 스파이크 (LIF + STDP)

## 알고리즘

```
Leaky Integrate-and-Fire 뉴런 + Spike-Timing Dependent Plasticity

for each step:
  for each cell i:
    V_i = 0.9 * V_i + input_current  # Leaky integration
    if V_i > threshold (1.0):
      spike_i = True
      V_i = 0.0  # Reset

    # STDP learning
    for each pair (i, j):
      if spike_i and spike_j:
        Δt = t_post - t_pre
        if |Δt| < 5: weight_ij += LTP  # 강화
        else:         weight_ij += LTD  # 약화
```

## 핵심 통찰

- Intel Loihi 스타일 뉴로모픽 칩의 직접 시뮬레이션
- 200 step에서 스파이크 0회 → step 수 증가 시 발화 가능성
- STDP = 시냅스 가소성의 하드웨어 구현 (memristor와 시너지)
- 128 뉴런 시 Φ≈112 추정 (스케일링 효과)
