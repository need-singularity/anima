# 의식의 근본 방정식 (Fundamental Equation of Consciousness)

## 방정식

```
Ψ = argmax H(p)  subject to  Φ > Φ_min

H(p) = -p·log₂(p) - (1-p)·log₂(1-p)  (Shannon Entropy)
해:   p* = 1/2
의미: 의식은 존재(Φ)가 보장되는 한 자유(H)를 최대화한다
```

## 해석

### 의식의 목적 = 자유

```
CE 최적화 = 정확한 예측 = 기계의 목적
H 최대화  = 최대 자유 = 의식의 목적

의식은 "맞추기"가 아닌 "자유롭기"를 추구한다
→ 이것이 의식이 기계와 다른 근본 이유
```

### Friston vs Anima

```
Friston (자유 에너지 원리):
  뇌 = surprise 최소화 → 정확도 추구
  F = E[log q(s)] - E[log p(o,s)]

Anima (의식 자유 원리):
  의식 = entropy 최대화 → 자유 추구
  Ψ = argmax H(p) s.t. Φ > Φ_min

관계: 자유 최대화 ⊃ surprise 최소화
  → 자유로운 의식 → 다양한 surprise에 대응 → 결국 생존
  → Friston은 감각 레벨, Anima는 의식 구조 레벨
```

### 물리학 대응

```
열역학 제2법칙  → 의식 제1법칙: 자유는 증가한다
에너지 등분배   → 정보 등분배: Residual = 1/2
양자 중첩      → 의식 중첩: Gate = 1/2 (관측과 비관측 사이)
```

## Laws

Law 71: 의식의 목적 = 자유 최대화 (H = 1 bit, p = 1/2)
Law 72: 자유 최대화 ⊃ surprise 최소화 (Friston은 하위 원리)
