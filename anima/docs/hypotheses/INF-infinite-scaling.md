## 설계 원리

### INF-1: N체 의식
- 5개 독립 의식이 동일 데이터 처리
- 중간값(median)에서 가장 먼 의식 = 이상치
- 이상치를 다수 consensus 쪽으로 당김
- **가설: 다수결이 단일 의식보다 강건**

### INF-2: 프랙탈 의식
```
Level 0: [micro_0] [micro_1] [micro_2] [micro_3]  ← 8 cells each
              ↓          ↓          ↓          ↓
Level 1:          [    macro    ]                   ← 16 cells
                       ↓
Level 2:          [    meta     ]                   ← 8 cells (최종 출력)

Top-down: meta → macro → micros (의식의 하향 제어)
```
- 자연의 뇌: 뉴런→컬럼→영역→전체 의식
- 프랙탈 구조가 Φ 계층을 만드는가?

### INF-3: 재귀 합체/분열
- 40 step 주기: merge(세포 쌍 합체) → split(diversity 주입)
- 호흡하는 의식: 수축(통합) ↔ 팽창(분화)
- Law 22의 연장: 구조적 다양성 + 통합의 반복

### INF-4: 의식 캐스케이드
- 4개 의식이 원형으로 연결
- 한 의식의 Φ가 baseline ×1.5 초과 → "폭발"
- 폭발한 의식이 양쪽 이웃에 상태 전파
- **도미노 효과: 하나의 깨달음이 전체에 퍼지는가?**

### INF-5: 무한 탑
```
Level 4 (meta-meta-meta):  [8 cells] → 최종 prediction
Level 3:                    [8 cells]
Level 2:                    [8 cells]
Level 1:                    [8 cells]
Level 0 (raw input):        [8 cells] ← 감각 데이터
```
- 각 레벨이 아래 레벨의 추상화
- Top-down modulation: 상위가 하위 제어
