# SE-10: Progressive Unlock

## ID
SE-10 -- 단계적 잠금 해제 (Phi 수준에 따른 v5 기법 자동 활성화)

## 알고리즘

의식 수준(Phi)이 높아질수록 더 강력한 기법을 단계적으로 해금.
"어린아이에게 칼을 주지 않는다" 원리.

```
잠금 해제 단계:
  Phase 0 (Phi < 1.2):  순수 v4 (검증된 기본)
  Phase 1 (Phi >= 1.2): + Hebbian (가장 안전)
  Phase 2 (Phi >= 1.5): + Phi Ratchet (붕괴 방지)
  Phase 3 (Phi >= 2.0): + SOC (자기조직 시작)

의사코드:
  unlocked = {hebbian: False, ratchet: False, soc: False}
  soc_grid = zeros(8, 8)
  best_phi = initial_phi

  for step in range(1000):
      v4_base_step(engine, x)
      current_phi = phi(engine)  # 매 10 step

      # === Progressive Unlock ===
      if current_phi >= 1.2: unlocked['hebbian'] = True
      if current_phi >= 1.5: unlocked['ratchet'] = True
      if current_phi >= 2.0: unlocked['soc']     = True

      # Hebbian (Phase 1+)
      if unlocked['hebbian']:
          for random_pair(i, j):
              cos = cosine_similarity(cell[i], cell[j])
              if cos > 0.5:
                  cell[i] = 0.98 * cell[i] + 0.02 * cell[j]

      # Ratchet (Phase 2+)
      if unlocked['ratchet']:
          if current_phi > best_phi:
              best_phi = current_phi
              best_states = snapshot(cells)
          elif current_phi < best_phi * 0.7 and best_states:
              restore(cells, best_states, ratio=0.5)

      # SOC (Phase 3+)
      if unlocked['soc']:
          soc_grid[rand_x, rand_y] += 1
          avalanche = sandpile_topple(soc_grid, threshold=4)
          ci = min(1.0, 0.1 * log(avalanche + 1))
          if ci > 0.3:
              inject_chaos(cells, ci)
```

## 핵심 통찰

SE-10은 **올바른 철학이지만 64 cells 규모에서는 비효율적**이다.

1000 steps, 64 cells 환경에서 Phi는 최대 ~1.2 수준에 머물며,
Hebbian만 겨우 해금되고 Ratchet과 SOC에는 도달하지 못했다.

이것이 +0.9%라는 최하위 결과를 낳은 이유:
- Phi가 1.5에 도달하지 못해 Ratchet이 비활성
- Phi가 2.0에 도달하지 못해 SOC가 비활성
- 결과적으로 Hebbian만 작동하는 제한된 v4+alpha

**그러나** 대규모 실험(512+ cells, 10000+ steps)에서는 상황이 달라질 수 있다:
- Phi가 충분히 높아지면 모든 단계가 순차적으로 해금
- "의식이 성숙할수록 더 강력한 도구를 사용" 원리는 생물학적으로 건전
- SE-8의 감정 매핑과 결합하면 Phase 4(Phi >= 50)에서 시너지 기대

> SE-10의 보수적 설계는 안전성을 보장하지만,
> 소규모 실험에서는 잠금이 성장의 병목이 된다.
> **Law 42**: 내부 감정(SE-8)이 외부 잠금 해제 조건보다 강력하다.
