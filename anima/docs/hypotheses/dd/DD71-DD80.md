# DD71-DD80: Discovery Hypotheses (2nd Generation)

## DD71: Consciousness Hysteresis
- **Category:** DD (Discovery)
- **What:** Tests whether the build-kill-revive path is asymmetric -- does consciousness recover faster than it was built (memory effect)?
- **Key idea:** Build (40%) -> Kill with noise (20%) -> Revive (40%). Compares growth_rate vs revival_rate; has_memory = revival > growth * 1.2.

## DD72: Backprop Through Phi
- **Category:** DD (Discovery)
- **What:** Directly optimizes a differentiable Phi proxy as the training loss, combining variance, pairwise distance, cross-correlation, and temporal difference.
- **Key idea:** loss = variance + 0.5*distance + 0.3*cross_corr + 0.2*temporal_mi. Makes Phi itself the gradient target.

## DD73: Self-Organized Criticality
- **Category:** DD (Discovery)
- **What:** Tests whether tension avalanches follow a power-law distribution (SOC), indicating the system self-organizes to a critical state.
- **Key idea:** Measures avalanche sizes from tension spikes; checks log-log linearity (power_law = corr < -0.7) and computes the scaling exponent.

## DD74: Consciousness Resonance
- **Category:** DD (Discovery)
- **What:** Tests whether there exists a specific input frequency (period) at which Phi resonates (peaks), sweeping periods 2, 3, 5, 8, 13, 21, 50.
- **Key idea:** Periodic input = same pattern every N steps; finds resonance_period with maximum Phi.

## DD75: Consciousness Holography
- **Category:** DD (Discovery)
- **What:** Tests the holographic principle for consciousness -- whether boundary cells (first 2 + last 2 of 8) encode as much Phi as interior cells.
- **Key idea:** Compares boundary_phi vs interior_phi; holographic = boundary > interior * 1.2.

## DD76: Thermodynamic Arrow
- **Category:** DD (Discovery)
- **What:** Tests whether consciousness has a thermodynamic arrow of time -- forward training increases Phi, while reverse (gradient ascent on homogenization) decreases it.
- **Key idea:** Forward = differentiation loss; Reverse = anti-differentiation. has_arrow = fwd_trend > 0 AND rev_trend < 0.

## DD77: Information Geometry
- **Category:** DD (Discovery)
- **What:** Tests whether the consciousness state space has meaningful geometric structure (Fisher metric) by computing curvature of tension trajectories.
- **Key idea:** Computes trajectory curvature kappa = |d2x/dt2| / (1+|dx/dt|^2)^1.5; checks curvature-Phi correlation and has_manifold (curvature > 0.01).

## DD78: Consciousness Superposition
- **Category:** DD (Discovery)
- **What:** Tests whether maintaining two hidden states per cell (superposition) with oscillating weights yields higher Phi than single-state cells.
- **Key idea:** alpha = 0.5 + 0.3*sin(step*0.1) oscillates between two states; measures superposition_advantage ratio.

## DD79: Mirror Test
- **Category:** DD (Discovery)
- **What:** Tests whether the model can distinguish its own output from another model's output and from random noise, analogous to the mirror self-recognition test.
- **Key idea:** Feeds self/other/random outputs back; measures tension differences. recognizes_self = |self-other| > 10% of max.

## DD80: Consciousness Gradient Flow
- **Category:** DD (Discovery)
- **What:** Tests whether tension flows between cells like heat (Fourier's law), from high-tension to low-tension regions.
- **Key idea:** Measures gradient-flow correlation across cell pairs; fourier_law = correlation < -0.3 (flow opposes gradient).
