## DD11: Klein Bottle

- **Function**: `run_DD11_klein_bottle`

- **Mechanism**: Non-orientable manifold: each cell connected to ALL others with alternating twist signs. twist = -1 if (i+j)%2 == 1, else +1. Blend ratio 0.85/0.15. Creates rich symmetric/antisymmetric interaction patterns.
