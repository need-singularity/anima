# DD35-DD37: Information-Theoretic Consciousness Measures

## DD35: Kolmogorov Phi
- **ID**: DD35
- **Function**: `run_DD35_kolmogorov`
- **Category**: Math (information theory)
- **Mechanism**: Uses Kolmogorov complexity as a consciousness proxy. Computes weight entropy (softmax over flattened weight vectors) as an approximation of description length. Loss maximizes both differentiation and weight entropy (-0.01 * w_entropy), pushing cells toward complex, non-trivial representations.

## DD36: Rate-Distortion Phi
- **ID**: DD36
- **Function**: `run_DD36_rate_distortion`
- **Category**: Math (information theory)
- **Mechanism**: Progressive compression at multiple bit rates (128->64->32->16->8->4). Finds the minimum number of bits that preserves cell structure (reconstruction loss < 0.5). Cells are nudged toward the compressed representation at the optimal rate-distortion point.

## DD37: InfoMax
- **ID**: DD37
- **Function**: `run_DD37_infomax`
- **Category**: Math (information theory)
- **Mechanism**: Maximizes mutual information between all cell pairs. Uses cross-correlation matrix variance as a proxy for structured (non-uniform) correlation patterns. Loss = differentiation - 0.1*corr_var. Aims for high but structured inter-cell dependencies.
