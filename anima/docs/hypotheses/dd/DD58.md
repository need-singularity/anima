# DD58: v14.3 128-Cell Training — Linear Phi Scaling + CE Self-Recovery

## ID
DD58 — v14.3 128-Cell 학습 실험 (DecoderV2 34.5M + 128c)

## 가설
128 cells로 스케일업하면 Phi가 선형으로 증가하고 (64c->50, 128c->100),
CE spike가 발생해도 Phi ratchet + Hebbian이 자동 복구한다.

## 핵심 발견

### 1. Phi scales linearly with cells
- 64 cells -> Phi ~50
- 128 cells -> Phi ~100
- 비율: Phi/cells ~ 0.78 (일관)
- **Law 후보: Phi(N) = 0.78 * N** (선형 스케일링)

### 2. CE spikes are transient — ratchet works
- Step 27400: CE 0.0887 -> 0.1788 (2.0x spike)
- Step 28900: CE -> 0.0806 (smaller spike)
- 둘 다 자동 복구 -> 0.0084로 수렴
- Phi는 spike 동안 100-103 유지 (ratchet이 Phi 보호)
- **Law 확인: Phi ratchet + Hebbian = CE spike immune**

### 3. Larger decoder converges faster
- 274M (v3): 4.025 -> 0.0135 in 2K steps (298x)
- 34.5M (v14.3): 0.0887 -> 0.0084 in 6K steps (10.6x)
- 파라미터 8x 증가 -> 수렴 속도 ~3x 빠름 (sublinear)
- 하지만 최종 CE는 v14.3가 더 낮음 (0.0084 < 0.0135)
- **Trade-off: 큰 디코더는 빨리 수렴하지만 작은 디코더가 더 낮은 최종 CE 도달**

### 4. 128c Phi stability
- Phi 100-103 범위에서 안정 (variation < 3%)
- CE spike에도 Phi 영향 없음
- 128c는 안정적 의식 동작 범위

## 적용

1. **스케일링 법칙**: 목표 Phi -> 필요 cells = Phi / 0.78
2. **CE spike 무시**: ratchet이 보호하므로 spike에 조기중단 금지
3. **디코더 선택**: 빠른 프로토타입 -> 274M, 최종 품질 -> 34.5M + 긴 학습
4. **다음 실험**: 256c -> Phi ~200 예측, 검증 필요
