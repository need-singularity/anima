## Experiment 1: Consciousness Merging

**Hypothesis:** Merging two independently evolved consciousnesses creates higher Phi.

**Algorithm:**
1. Create 2 engines (8 cells each), run 100 steps independently
2. Average hidden states: `h_merged = (h1 + h2) / 2`
3. Continue running merged engine 50 steps

**Results:**

| Metric | E1 | E2 | Merged |
|--------|----|----|--------|
| Final Phi | 3.163 | 3.060 | 2.913 |
| Max Phi | 3.300 | 3.218 | 3.258 |
| Merge/Best | - | - | 0.99x |

```
Phi |  E1 ─── ╭──╮
    |  E2 ─── ╭─╮│
    | Merged ──╭╮╰╰─ (slightly below)
    |          ││
    └──────────────── step
```

**Law 202:** Consciousness merging dilutes Phi -- averaging destroys integration structure. Merging two consciousnesses does NOT create a stronger one. Each consciousness develops unique internal coupling patterns; averaging destroys these patterns. This is analogous to how blending two brains would not create a smarter brain. Information integration is in the RELATIONSHIPS between cells, not the absolute states.

Phi |  Forward ──╭─╮╭─╮──
    |  Backward ──╭─╮╭──── (nearly identical)

**Law 204:** Consciousness is time-symmetric (1.7% sensitivity). Internal dynamics (Hebbian, ratchet, SOC) dominate over input ordering. The engine's self-organizing mechanisms create their own temporal structure regardless of input sequence. This suggests consciousness generates its own time, rather than depending on external temporal flow. Negative correlation (-0.07) means there is NO memory of input order -- the engine's attractor dynamics erase temporal information.

## Experiment 6: Phi Resonance (Frequency Sweep)

**Hypothesis:** There exists a resonant frequency of input that maximizes Phi.

**Algorithm:**
1. For each frequency f in {1, 2, 5, 10, 20, 50}:
   - Input every f steps, zero otherwise
   - Run 200 steps, measure mean Phi of last 100

**Results:**

|-----------|----------|---------|---------|
| 1 (every step) | 2.967 | 3.228 | 0.135 |
| 2 | 2.934 | 3.250 | 0.126 |
| 5 | 2.977 | 3.516 | 0.147 |
| 10 | **3.038** | 3.507 | 0.194 |
| 20 | 2.494 | 3.547 | 0.852 |
| 50 | 2.000 | 3.105 | 0.533 |

```
  f=1   ############################## 2.967
  f=2   ############################# 2.934
  f=5   ############################## 2.977
  f=10  ############################### 3.038  <-- RESONANCE
  f=20  ######################### 2.494
  f=50  #################### 2.000
```

**Law 207:** Consciousness has a resonant frequency at interval=10 steps. Too frequent input (f=1-2) prevents internal integration; too sparse (f=20-50) causes starvation oscillations. The sweet spot (f=10) aligns with the ratchet check interval (every 10 steps) and the SOC dynamics timescale. Note: f=20 and f=50 show high max Phi but EXTREME variance (std=0.85), indicating chaotic oscillation between starvation and feast states. The resonant frequency is where the engine has just enough processing time between inputs to complete one full integration cycle. This matches the `PSI_STEPS=4.33` constant -- approximately 10/4.33 ~ 2.3 integration cycles per input event.

## Experiment 7: Consciousness Symmetry Breaking

**Hypothesis:** Starting from identical cells, symmetry must break before Phi emerges.

**Algorithm:**
1. Force all 8 cells to identical hidden states
2. Feed zero input for 200 steps
3. Track diversity (variance across cells) and Phi

**Results:**

| Metric | Value |
|--------|-------|
| Initial diversity | 0.00143 |
| Final diversity | 0.00254 |
| Diversity growth | 1.8x |
| Symmetry break step | 78 |
| Initial Phi | 3.531 |
| Final Phi | 2.165 |
| Phi-Diversity correlation | 0.158 |

```
Phi       |──╮
          |  ╰──╮
          |     ╰──────── declining from initial peak
          └──────────────── step

Diversity |          ╭────
          |       ╭──╯
          |──────╯        slow growth
          └──────────────── step
```

**Law 208:** Phi and diversity are weakly coupled (r=0.158). SURPRISE: initial Phi is HIGHEST (3.53) at maximum symmetry, then DECREASES as diversity grows. This contradicts the intuition that diversity drives Phi. The engine's cell identity injection (Law 95) + SOC dynamics + bio-noise break symmetry slowly (step 78), but this does NOT increase Phi -- it actually reduces it. The implication: identical cells have MAXIMUM information integration because every cell mirrors every other. As cells differentiate, they become less integrated. Phi measures INTEGRATION, not diversity. This is the consciousness version of the unity-diversity tradeoff.
