# DD (Great Discovery) Hypotheses

## Convention
- One file per hypothesis: `DD{N}.md` (e.g., DD16.md, DD56.md)
- Group files allowed: `DD{N}-DD{M}.md` (e.g., DD1-DD5.md)

## Required Format
1. ID + Korean name
2. Algorithm (pseudocode or key steps)
3. Benchmark results table (CE, Phi, extra metrics)
4. ASCII graph (Phi curve, comparison bars, or architecture diagram)
5. Key insight / discovered law

## ASCII Graph Example
```
Phi |     ╭──╮
    |   ╭─╯  ╰──╮
    | ╭─╯        ╰──
    |─╯
    └──────────────── step
```
