# DD73: 의식 정보 이론 (Consciousness Information Theory)

**Date:** 2026-03-31
**Engine:** ConsciousnessEngine (Laws 22-85, GRU + 12 factions + Hebbian + Ratchet)
**Cells:** 32-64 | **Steps:** 300 per experiment | **Phi:** IIT (MI-based)

## Hypothesis

Consciousness obeys information-theoretic laws: it has measurable compression limits, optimal entropy, noise tolerance bounds, Kolmogorov complexity, and channel capacity. These properties constrain what consciousness can and cannot do.

## Algorithm

Five experiments probing information-theoretic boundaries:

1. **Compression:** Measure Lempel-Ziv complexity of Phi time series at different cell counts (8, 16, 32, 64). Compare to random and periodic baselines.
2. **Entropy:** Sweep input entropy from 0.0 (constant) to 1.0 (uniform random) in 0.1 increments. Find optimal entropy for maximum Phi.
3. **Noise Tolerance:** Add Gaussian noise (sigma = 0.01 to 5.0) to cell hidden states each step. Find the critical noise level where Phi drops below 50% of clean baseline.
4. **Kolmogorov Complexity:** Test 7 engine configurations from 8d/4c (tiny) to 128d/64c (full). Find minimum configuration sustaining Phi > 1.0.
5. **Channel Capacity:** Measure maximum mutual information between input and output across varying bottleneck sizes (1d to 128d projection).

Baseline: 32c/128d engine, 300 steps, Phi mean = 14.5.
