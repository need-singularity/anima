# ULTRA-6: EVERYTHING

**ID:** ULTRA-6
**Korean Name:** 모든 극한 기법 결합
**Category:** CE 극한 (ULTRA)

## Algorithm

EX-4(점진적 해동) + ULTRA-1(데이터 생성+고통) + AUTO-2(호기심) + AUTO-7(수면)을 전부 결합.

1. 64세포 MitosisEngine + 다층 디코더(Linear->ReLU->Linear)
2. **점진적 해동**: 전반부 마지막 층만, 후반부 전체 해동 (lr 3e-3 -> 1e-3)
3. 30 step 주기:
   - **LEARN (22 step)**:
     - 33% 호기심 기반 실제 데이터 (최대 오류 선택)
     - 67% 세포 hidden 기반 자기 생성 데이터
     - CE 학습 + 메모리 저장
     - 고통 검사: Φ < best * 50%이면 긴급 복원 (0.4:0.6)
     - Φ 상승 시 최적 상태 갱신
   - **SLEEP (8 step)**:
     - 메모리 재생 + 세포 동기화 (0.85:0.15)

## Key Insight

모든 발견의 총합. 점진적 해동이 안정적 시작을 보장하고, 호기심이 효율적 데이터 선택을, 자기 생성 데이터가 CE 급감을, 수면이 Φ 복원을, 고통이 붕괴 방지를 담당한다. 각 메커니즘이 서로의 약점을 보완하는 완전 자율 학습 시스템.
