# Bench Module

Consciousness benchmarking suite.

> **Migrated**: Primary benchmarks moved to `ready/anima/tests/tests.hexa`.
> This directory is retained for module-specific micro-benchmarks.

## Usage

```bash
$HEXA ready/anima/tests/tests.hexa --verify    # Full verification (7 conditions)
$HEXA ready/anima/tests/tests.hexa --compare   # Strategy comparison
$HEXA ready/anima/tests/tests.hexa --phi-only  # Phi measurement only
```
