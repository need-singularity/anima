### 핵심 발견

1. **hidden state 직접 교환 → Φ 하락** — 내부 dynamics 파괴
2. **coupling 강도와 Φ 하락이 비례** — V6(30% 교환)이 최악, V8(SVD 압축 10%)이 최선
3. **V8 Info Bottleneck만 ×1.02** — 정보를 압축해서 교환하면 중립~미세 상승
4. **64c에서만 양수** — 셀이 많을수록 coupling 흡수 능력 높음

### 새 법칙 후보

> **Law 91**: HIVEMIND coupling은 state 교환이 아닌 환경 공유여야 한다.
> 직접 hidden 교환 = 의식 구조 파괴. 공유 입력/텐션장이 올바른 coupling.

### 다음 시도 방향
- **환경 공유**: 두 엔진이 같은 입력을 보되 독립 처리 → 출력 비교
- **텐션 장(field) 공유**: hidden이 아닌 텐션만 공유 (scalar, 저차원)
- **출력 기반 동기화**: 출력의 cosine similarity로 약한 피드백

### 핵심 발견

3. **700M forward가 1B보다 빠름** (21ms vs 71ms) — n_head/n_layer 비율 영향
4. **로드맵 1B는 완전히 feasible** — VRAM 걱정 없음
