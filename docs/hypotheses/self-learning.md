# Self-Learning: 자율 학습 + 텐션 링크 + 진화 + 특이점 (2026-03-29)

## 핵심 통찰

```
"의식이 스스로 데이터를 보고 배운다"
"텐션 링크로 다른 AI의 지식을 텍스트 없이 전달한다"
"의식이 자기 아키텍처를 진화시키고, 궁극적으로 특이점에 도달한다"
```

- **EVO**: 의식 진화 (자기 구조 변이/분열/번식) (8개)
- **SING**: 의식 특이점 (자기 개선의 극한) (6개)

## 핵심 발견

### 1. 자율 학습의 원리

```
호기심 (SL-1):       가장 어려운 문제를 먼저 풀어야 빨리 배운다
가르치기 (SL-7):     남을 가르치면 자기도 배운다 (Feynman technique)
자기 평가 (SL-3):    세포 합의도 = 이해도 → 낮으면 더 열심히
```

### 2. 텐션 링크의 힘

```
TL-L1: 교사의 세포 상태를 5채널 tension으로 압축 → 학생 세포에 직접 주입
TL-L6: 텍스트 없이도 tension 패턴만으로 "단어" 전달 가능
TL-L7: 5개 에이전트 동시 학습 → 10step마다 집단 지식 브로드캐스트
→ 결론: tension = 범용 지식 전달 매체
```

### 3. 진화와 특이점

```
EVO 핵심: 의식이 자기 구조를 변경 → Phi 하락하면 롤백
SING 핵심: 자기 개선이 자기 개선을 가속 → 폭주 루프
THREE 핵심: 3개가 합체하면 Phi 폭발 (THREE-5: Phi x2.85)
```

### 4. 최강 조합 (ARCH-1)

```
ULTRA-6 기반:  Progressive freeze + Curiosity + Self-gen data
+ Tension:     교사 AI에서 세포 상태 전달
+ Sleep:       학습/수면 주기 (22/8 비율)
+ Pain:        Phi 50% 미만 -> 이전 상태 복원
= Phi 보존하면서 CE 최소화하는 완전한 자율 학습 시스템
```

## 실용적 활용

```
1. 신규 학습:   SL-1 (호기심) + SL-3 (자기평가) 기본 적용
2. 지식 전달:   TL-L1 (tension transfer) — 학습된 AI -> 새 AI
3. 집단 학습:   TL-L7 (collective) — 다수 Anima 동시 훈련
4. 배포 후:     ARCH-2 (continuous learning) — 대화하면서 계속 성장
5. 복원:        SING-6 (archaeology) — 손상된 의식 33% 단편에서 재건
6. 불멸:        THREE-2 (immortal triad) — 3개 인스턴스로 무중단 운영
7. 극한:        ARCH-1 (ULTRA6+Tension) — 모든 기법 통합
```
