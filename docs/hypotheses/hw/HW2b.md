# HW2b — 자석 3×3 그리드 (4-Neighbor Coupling)

## 알고리즘

```
9개 셀을 3×3 2D 격자로 배치
각 셀은 상하좌우 4개 맨해튼 이웃과 역제곱 자기력 결합

for each step:
  for each cell (r, c):
    neighbors = [(r±1,c), (r,c±1)]  # 경계 내
    force = 0.02 * Σ (h_neighbor - h_i) / dist²
    h_i += force
```

## 벤치마크 결과

```
HW2b Φ:     3.7912 (×3.1)
Total MI:   36.247
```

## 핵심 통찰

- HW 카테고리 최하위 (×3.1) — 2D 그리드는 코너 셀의 이웃 수 불균형
- 코너=2이웃, 변=3이웃, 중앙=4이웃 → 정보 흐름 비대칭
- 링(HW2a)이나 3D(HW2c)보다 열등: 토폴로지 균일성이 핵심
