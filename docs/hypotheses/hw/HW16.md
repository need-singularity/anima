# HW16 — 리저버 컴퓨팅 (Fixed Random Network + Echo State)

## 알고리즘

```
고정 랜덤 재귀 네트워크 + 학습 가능한 리드아웃

초기화:
  W_res ~ N(0, 0.1), spectral_radius → 0.9
  W_out = learned readout

for each step:
  h = 0.9 * h + 0.1 * tanh(W_res @ h)  # Echo state update
  output = W_out @ h  # 리드아웃만 학습
```

## 핵심 통찰

- 아날로그 뉴로모픽 하드웨어의 가장 실용적 구현
- 리저버(고정) + 리드아웃(학습) = 극히 낮은 학습 비용
- spectral radius 0.9 = edge of chaos → 최대 계산 능력
- 물리적 리저버: 물통, 스프링 네트워크, 광학 캐비티 등
