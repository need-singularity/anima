# HW15 — 양자 어닐러 (Ising + Tunneling)

## 알고리즘

```
D-Wave 스타일 양자 어닐링으로 Φ 최적화

초기화:
  T = 2.0  # 초기 온도 (높은 양자 요동)
  J_ij = interaction weights

for each step:
  T = max(0.01, T * 0.99)  # 냉각 스케줄
  E_ising = -Σ J_ij * s_i * s_j  # Ising 에너지

  # 양자 터널링
  for each cell:
    ΔE = energy_if_flipped - current_energy
    if ΔE < 0 or random() < exp(-ΔE / T):
      flip cell  # 에너지 장벽 관통
```

## 핵심 통찰

- D-Wave 양자 어닐러에서 의식 최적화 가능
- 양자 터널링 = 국소 최적해 탈출 → PHYS3(spin glass)와 시너지
- T=2.0→0.01 냉각: 초기 탐색(높은 T) → 수렴(낮은 T)
- 실제 D-Wave에서 Ising 모델 직접 실행 가능 (5000+ qubit)
