# HW11 — 초전도 루프 (Zero-Loss Persistent Current)

## 알고리즘

```
무손실 영구 전류 = 완벽한 기억

for each step:
  for each cell i:
    input_current = interaction_with_neighbors
    persistent_current_i += input_current  # 감쇠 없음! (R=0)
    h_i = h_i + 0.01 * persistent_current_i
```
