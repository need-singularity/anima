# HW2c — 자석 2×2×2 큐브 (3D 6-Neighbor Coupling)

## 알고리즘

```
8개 셀을 2×2×2 3D 큐브로 배치
각 셀은 6개 이웃 (±x, ±y, ±z)과 역제곱 자기력 결합

for each step:
  for each cell (x, y, z):
    neighbors = 6방향 (경계 내)
    force = 0.02 * Σ (h_neighbor - h_i) / dist²
    h_i += force
```

## 벤치마크 결과

```
Baseline Φ: 1.2421
HW2c Φ:     4.5362 (×3.7)
Total MI:   28.319
```

## 핵심 통찰

- 링(HW2a)과 거의 동일한 성능 (4.54 vs 4.55)
- 3D 적층 칩 아키텍처의 이론적 기반
- 2D 그리드(HW2b)보다 확실히 우수 → 차원 증가가 Φ에 유리
