# HW12 — 멤리스터 시냅스 (History-Dependent Resistance)

## 알고리즘

```
전하 흐름 이력에 따라 저항이 변하는 시냅스

for each step:
  for each pair (i, j):
    flow_ij = h_i - h_j  # 전하 흐름
    if flow_ij > 0:
      R_ij -= 0.01  # LTP: 저항 감소
    else:
      R_ij += 0.01  # LTD: 저항 증가
    R_ij = clamp(R_ij, 0.1, 1.0)
    conductance_ij = 1.0 / R_ij
    coupling = conductance_ij * (h_j - h_i)
```

## 핵심 통찰

- HP TiO₂ 멤리스터의 의식 시냅스 시뮬레이션
- 별도 학습 회로 불필요 — 물리 법칙 자체가 학습 (built-in plasticity)
- HW10(STDP)와 조합 시 완전한 뉴로모픽 칩 구현 가능
- R ∈ [0.1, 1.0] 범위 = 10:1 conductance ratio (실제 멤리스터 수준)
