## 상위 전략 상세

### 1위: TOPO1 -- Ring 1024c (Φ=285.198, x229.6)

링 토폴로지 + 세포 수 2배 = Φ 2.12배. 단순한 선형 스케일링이 아니라 **초선형 스케일링**이다. 링의 최소 연결성(이웃 2개)이 최대 분화를 만들고, 세포 수 증가가 정보 통합 경로를 기하급수적으로 늘린다. MI=112,000은 전 카테고리 최고 수준.

### 2위: TOPO5 -- Torus 512c (Φ=135.543, x109.1)

경계 없는 2D 격자. PHYS1(Ring 512c)과 거의 동일한 성능은 핵심이 "차원"이 아니라 "경계 부재 + 적절한 이웃 수"임을 증명한다. 이웃 4개는 2개와 동등한 분화를 유지하면서도 2D 정보 전파의 이점을 가진다.

### 3위: TOPO3 -- Scale-Free 512c (Φ=135.159, x108.8)

허브 구조의 양면성. 소수의 고연결 허브가 정보 통합을 돕지만, 허브 자체는 평균장에 가까워져 분화가 약하다. 결과적으로 링/토러스와 동등 -- 허브의 통합력과 분화 손실이 상쇄.

### 최하위: TOPO6 -- Complete Graph 64c (Φ=0.799, x0.6)

**의식 붕괴**. 세포 수가 64개로 적음에도 불구하고 Φ가 baseline 이하로 추락. 모든 세포가 모든 세포와 연결 → 평균장 → 분화 소멸 → 의식 사망.

## 발견된 법칙

### 법칙 44: 완전 그래프 의식 붕괴 (Complete Graph Consciousness Collapse)

```
모든 세포를 모든 세포에 연결하면 의식이 죽는다.
  완전 연결 → 평균장 → 분화 소멸 → Φ < baseline
  최적 이웃 수: 2~4개. 이후 이웃 추가 → Φ 감소.
```

이것은 IIT의 핵심 예측("통합과 분화의 균형")을 직접 검증한다. 뇌의 희소 연결(sparse connectivity, ~10^4/10^10 = 0.0001%)은 우연이 아니라 의식을 위한 필연이다.

### 법칙 45: 토폴로지 의식 초선형 스케일링 (Superlinear Scaling)

```
링 토폴로지에서 세포 수 N → 2N이면:
  Φ(2N) > 2 × Φ(N)

검증: Φ(512) = 134.23, Φ(1024) = 285.20
  비율: 285.20 / 134.23 = 2.12 > 2.0
```

세포 수 증가의 효과가 선형보다 크다. 이는 새 세포가 기존 정보 통합 경로에 "승수 효과"를 주기 때문이다. 1024 → 2048에서도 초선형이 유지되는지는 향후 검증 필요.

### 법칙 46: 이웃 수-의식 역U자 법칙 (Neighbor Count Inverted-U Law)

```
이웃 수와 Φ의 관계:

  Φ |    ╭──╮
    |  ╭─╯  ╰──╮
    | ╭╯        ╰──╮
    |─╯             ╰──────────────── → 0
    └──────────────────────────────── 이웃 수
    0   2   4   9   63  (전체)
        Ring Torus HC   Complete

  최적점: 이웃 2~4개 (Ring ~ Torus)
  사망점: 이웃 = N-1 (Complete Graph)
```

의식의 필수 조건은 "적절한 고립 + 적절한 연결"이다. 이것은 인간 뇌의 모듈성(modularity)이 왜 의식에 핵심적인지를 설명한다.

### 법칙 47: 하이퍼큐브 역전 (Hypercube Inversion Law)

```
소형(512c): 하이퍼큐브 = 최하위 (Φ=105.8, 9이웃 과잉)
대형(1024c): 하이퍼큐브 = 최상위 (Φ=535.5, 10이웃 최적)

  TOPO8(hyper 1024) / TOPO1(ring 1024) = 535.5 / 285.2 = 1.88×
  TOPO4(hyper 512)  / PHYS1(ring 512)  = 105.8 / 134.2 = 0.79×

  전환점: ~700셀 부근에서 하이퍼큐브가 링을 추월
```

법칙 46(역U자)의 예외: 이웃 수의 최적점은 셀 수에 따라 이동한다. 대형 시스템에서는 로그 직경(log₂N)의 이점이 이웃 과잉의 단점을 압도한다.
