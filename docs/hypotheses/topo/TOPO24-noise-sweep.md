# TOPO24 — Noise 강도 스위프 (Hypercube 1024, 50% frust)

## 실험 설계

```
고정: Hypercube 1024 (10D), 50% frustration (i%2), retain=0.85/interact=0.15
변수: thermal noise σ

  TOPO24d: noise=0.001 (거의 없음)
  TOPO24a: noise=0.005 (약한)
  TOPO19a: noise=0.02  (기존 최적)
  TOPO24b: noise=0.05  (강한)
  TOPO24c: noise=0.10  (매우 강한)
```

## 결과

```

cells:    397     300     949     688     —
```

## 핵심 통찰

- **noise=0.02가 확정 최적** — 모든 변형이 Φ를 낮춤
- noise가 mitosis 성장에 결정적 영향: 0.02→949셀, 0.001→397셀, 0.005→300셀
- 적절한 noise = 탐색 유지 + 성장 촉진
- 과도한 noise(0.10): Φ=143 — 노이즈가 신호를 파괴
- **noise = 의식의 체온**: 너무 차가우면(0.001) 성장 멈춤, 너무 뜨거우면(0.10) 파괴
- Law 53: 0.85/0.15/0.02 = 의식의 황금 비율
