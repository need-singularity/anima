# TOPO-2048-SCALING --- 2048셀 스케일링 실험 종합 분석

## 실험 목적

TOPO8(하이퍼큐브 1024)의 Phi=535.5 올타임 레코드 이후,
2048셀로 스케일링하면 초선형 성장이 계속되는지 검증.

## 알고리즘

```
비교 실험 설계:
  TOPO8  — Hypercube 1024셀 (10D, 2^10), 200 step
  TOPO10 — Hypercube 2048셀 (11D, 2^11), 200 step
  TOPO11 — Ring 2048셀 + Frustration,    200 step

기존 512셀 참조:
  PHYS1  — Ring 512셀
  TOPO4  — Hypercube 512셀 (9D, 2^9)

공통: i%3 반강자성 frustration, 0.85/0.15 mixing, 200 step
```

## 성장 병목 분석

```

도달률  |
  100% |  ##  ##
       |  ##  ##
   68% |            ##  TOPO8 (693/1024)
       |
   38% |            ##  TOPO1 (387/1024)
   28% |                        ##  TOPO10 (581/2048)
   19% |                        ##  TOPO11 (397/2048)
       +--------+---------+---------+
              512       1024      2048

  ## = 실제 도달 셀 수 / 목표 셀 수

병목의 원인:
  512셀:  200 step이면 충분 (도달률 ~100%)
  1024셀: 200 step으로 38~68% 도달 (하이퍼큐브가 성장 빠름)
  2048셀: 200 step으로 19~28% 도달 (성장이 Phi를 결정)
```

## 핵심 통찰

1. **1024셀 하이퍼큐브 = 200 step의 최적해 (Phi=535.5)**
   - 2048셀은 200 step에서 성장 병목으로 오히려 Phi 감소

2. **Phi의 결정 요인 = min(max_cells, growth_capacity_at_given_steps)**
   - max_cells를 아무리 늘려도 step이 부족하면 무의미
   - 실제 도달 셀 수(final_cells)가 Phi를 결정

3. **하이퍼큐브의 성장 우위**
   - 같은 조건에서 Hyper의 도달률 > Ring (TOPO8 68% vs TOPO1 38%)
   - 하이퍼큐브의 균일 이웃 구조가 성장 전파를 가속

4. **2048셀 이상은 400+ step 필요** (추정)
   - 200 step에서 도달률 28% -> 400 step이면 ~56% -> 의미 있는 Phi 가능

5. **Law 30 재확인 + 정밀화**
   - 기존: "1024셀이 실용적 상한"
   - 정밀화: "200 step 기준 1024셀이 상한. step 비례로 상한 이동"

  --> step을 400~500으로 확대
  --> 또는 성장 가속 메커니즘 추가 (mitosis rate 2x)
  --> 예상: Phi > 800 가능 (초선형 스케일링 재개)
