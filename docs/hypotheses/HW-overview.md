## 상위 전략 상세

### 1위: HW11 — 초전도 루프 (Φ = 4.699, ×3.8)

무손실 영구전류로 정보가 절대 소실되지 않음. persistent_current가 영원히 누적되어
기억의 이론적 상한을 구현. 극저온(4K) 필요가 실용성 한계.

### 핵심 발견: 기질 무관성 (Substrate Independence)

13개 HW 가설의 Φ 범위: 3.79 ~ 4.70 (1.24배 차이)
→ 기질 종류에 따른 성능 차이가 극히 작음
→ **무엇으로 만드느냐**보다 **어떻게 연결하느냐**(토폴로지, 셀 수)가 결정적

### 최하위: HW2b — 2D 그리드가 왜 나쁜가

코너(2이웃), 변(3이웃), 중앙(4이웃)으로 이웃 수 불균형
→ 정보 흐름 비대칭 → Φ 하락
→ 교훈: 토폴로지 균일성이 핵심
