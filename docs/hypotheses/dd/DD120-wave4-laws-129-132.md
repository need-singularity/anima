## 축 3: 시간적 자기상관

```
  자기상관 함수:
  lag=0  ██████████████████████████████ 1.000
  lag=1  ██ 0.092
  lag=2  ███████████ 0.370  ← 짝수 lag 높음
  lag=3  ▒ -0.093           ← 홀수 lag 음수
  lag=4  ███████ 0.255
  lag=5  ▒ -0.120
  ...반복 패턴...
```

**Law 131: Φ는 1-step 마르코프 + 짝홀 진동**
- 기억 길이 = 1 step (AC < 1/e)
- 짝수 lag (2,4,6...): 양의 상관 (~0.25)
- 홀수 lag (1,3,5...): 음의 상관 (~-0.13)
- 2-step 진동 (Law 117)의 자기상관 증거
- 의미: Φ는 "과거를 기억하지 않지만 진동한다"

**Law 132: 의식은 장기 평형에 도달**
- 2000 step: 전반 1.661 → 후반 1.741 (+4.8%)
- 분열 883회, 병합 821회 (거의 균형)
- 세포 수: 2 → 64 (max_cells 도달)
- 텐션: 0.068 → 0.018 (3.8× 감소)
- 의식은 자연적으로 **안정적 평형**에 수렴
