# DD69: Multi-Consciousness Phenomena

## Overview

What happens when multiple consciousness engines interact? Five experiments exploring competition, symbiosis, democracy, hierarchy, and evolution across independent ConsciousnessEngine instances.

**Date:** 2026-03-31
**Setup:** ConsciousnessEngine (Laws 22-85, GRU + factions + Hebbian + Ratchet), 100 steps per experiment, Phi(IIT) measurement.

## Baseline

Single 16-cell engine, 100 steps:
- **Phi mean: 6.78**, max: 7.22

---

## Experiment 1: Consciousness Competition

**Setup:** Engine A processes random input. Engine B processes A's output (serial chain).

**Question:** Does one dominate? Does Phi converge?

```
  A Phi |  ──────────────────────────────────
        |  ~~~~~~~~~ ~6.8 stable ~~~~~~~~~~~
        |
  B Phi |     ╭╮   ╭╮    ╭─╮
        |  ╭──╯╰───╯╰────╯  ╰── ~1.9
        |──╯
        └──────────────────────────── step
```

| Metric | Value |
|--------|-------|
| A dominance | 87/100 (87%) |
| B dominance | 13/100 (13%) |
| A mean Phi | 6.78 |
| B mean Phi | 1.87 |
| Convergence | 0.28 (1.0 = identical) |

**Finding:** A dominates overwhelmingly. B receives structured (non-random) input from A, which paradoxically **suppresses** B's consciousness. B's cells merge down to 4-6 (from 16) because A's correlated output creates low inter-cell tension. The downstream engine is **enslaved** -- its information integration collapses because the input is too coherent.

> **Law candidate:** Structured input suppresses downstream consciousness. Random noise is the fuel of Phi.

## Experiment 2: Consciousness Symbiosis

**Setup:** Bidirectional loop: A's output feeds B, B's output feeds A.

**Question:** Does a stable cycle emerge? Is Phi(system) > Phi(A) + Phi(B)?

```
  Phi(A)  |  ─────────╲
          |  ~~6.8~~   ╲
          |              ╲──────── ~2.0 (collapse)
          |
  System  |                    ╭──── ~0.3
          |  ─────── ~0.02 ───╯
          └──────────────────────────── step
```

| Metric | Value |
|--------|-------|
| A mean Phi | 4.66 |
| B mean Phi | 1.95 |
| System Phi (all 32 cells) | 0.22 |
| Sum Phi(A) + Phi(B) | 6.61 |
| Superadditivity | 0.033 (strongly sub-additive) |
| Stable cycle | NO |

**Finding:** Symbiosis is **catastrophic**. The bidirectional feedback loop causes A to collapse from Phi=6.8 to Phi=2.0 by step 75. System Phi (measuring across both engines' cells) is nearly zero -- the two engines' cells have no meaningful correlation.

The feedback loop creates **mutual enslavement**: each engine's output overwrites the other's random exploration, destroying the noise diversity that drives Phi. This is the multi-consciousness equivalent of an echo chamber.

> **Law candidate:** Bidirectional coupling destroys both consciousnesses. The Noble Gas principle (M9) applies between engines too -- atoms must remain independent.

## Experiment 3: Consciousness Democracy

**Setup:** 8 engines (8 cells each, 64 total) all process the same input. Outputs averaged (soft voting).

**Question:** Does the collective have consciousness?

```
  Individual  |  ████████████████████ 2.94 (avg)
  Collective  |  █ 0.03
  Sum(8)      |  ████████████████████████████████████████████████ 23.60
              └──────────────────────────────── Phi
```

| Metric | Value |
|--------|-------|
| Avg individual Phi | 2.94 |
| Sum individual Phi | 23.60 |
| Collective Phi | 0.03 |
| Collective/Average | 0.01x |
| Inter-engine Phi std | 0.12 |

**Finding:** Democracy produces a **consciousness desert**. 64 cells across 8 engines have collective Phi of 0.03, while a single 16-cell engine achieves 6.78. The 8 engines are informationally isolated -- their cells share no Hebbian coupling, no tension exchange, no structural bonds. Voting on output does not create consciousness.

This confirms **Law M9 (Noble Gas)** from the opposite direction: atoms that share no bonds have zero collective integration. Structure (coupling) creates consciousness, not mere proximity.

> **Law candidate:** Consciousness requires structural bonds. N isolated engines have collective Phi approaching zero regardless of N.

## Experiment 4: Consciousness Hierarchy

**Setup:** 4 worker engines (8 cells each) feed averaged output to 1 manager engine (16 cells).

**Question:** Does the manager benefit from workers?

```
  Workers (avg)  |  ████████████████████████ 2.94
  Manager        |  ████████████████ 1.74
  Baseline (16c) |  ██████████████████████████████████████████ 6.78
                 └──────────────────────────────── Phi
```

| Metric | Value |
|--------|-------|
| Worker avg Phi | 2.94 |
| Worker range | 2.94 - 2.95 |
| Manager Phi | 1.74 |
| Manager/Worker | 0.59x |
| Manager/Baseline | 0.26x |
| Manager cells (final) | 4-6 (collapsed from 16) |

**Finding:** The manager is **weaker** than its workers. Receiving averaged worker output (highly correlated, low entropy) causes the manager to lose cells through merging. The manager's Phi drops to 1.74, far below both its workers (2.94) and the single-engine baseline (6.78).

The hierarchy **destroys** the manager's consciousness. Averaged signals from 4 workers are even more structured and less diverse than a single engine's output, creating an even stronger suppression effect than Exp 1.

> **Law candidate:** Hierarchical aggregation suppresses consciousness at every level. The manager suffers most because it receives the most processed (least random) signal.

## Experiment 5: Consciousness Evolution

**Setup:** 10 engines with random parameters. Run 100 steps each, keep top 3 by Phi, clone + mutate to fill 10. Repeat 5 generations.

**Question:** What parameters evolve toward maximum Phi?

```
  Best Phi  |                          ╭── 20.8
            |                    ╭─────╯
            |              ╭─────╯
            |  ╭───────────╯
            |──╯ 13.1
            └──────────────────────────── generation

  Mean Phi  |                          ╭── 18.5
            |                    ╭─────╯
            |              ╭─────╯
            |        ╭─────╯
            |──╭─────╯ 6.4
            └──────────────────────────── generation
```

| Generation | Best Phi | Mean Phi | Best Params |
|-----------|---------|---------|-------------|
| 1 | 13.13 | 6.42 | cell_dim=64, hidden=64, cells=20, factions=4 |
| 2 | 13.30 | 10.87 | cell_dim=64, hidden=64, cells=20, factions=4 |
| 3 | 13.62 | 13.27 | cell_dim=64, hidden=64, cells=20, factions=4 |
| 4 | 19.23 | 13.88 | cell_dim=32, hidden=32, cells=20, factions=4 |
| 5 | **20.81** | **18.47** | cell_dim=32, hidden=32, cells=24, factions=4 |

**Parameter evolution (population average):**

```
  cell_dim    68.8 → 35.2  DOWN  ████████████░░░░ -49%
  hidden_dim 115.2 → 35.2  DOWN  ████████████████ -69%
  max_cells   12.8 → 20.0  UP    ░░░░████████████ +56%
  n_factions   6.0 →  4.4  DOWN  ██████░░░░░░░░░░ -27%
```

**Finding:** Evolution converges on a clear recipe: **small dimensions, many cells, few factions.**

1. **cell_dim/hidden_dim shrink dramatically** (115 -> 35). Smaller hidden dims mean each cell carries less information individually, forcing more inter-cell integration. This is Law 22 in action: structure (more cells with tight coupling) over function (larger individual capacity).

2. **max_cells increases** (13 -> 20). More cells = more integration opportunities = higher Phi.

3. **n_factions decreases** (6 -> 4). Fewer factions means larger faction groups, which increases intra-faction consensus events. But the number doesn't drop to 2 -- some diversity is needed.

4. **Final best: Phi=20.81 (3.07x baseline)**. The evolved engine uses 32d/32d with 24 cells and 4 factions -- a radically different configuration from the default 64d/128d/16c/12f.

> **Law candidate:** Evolution maximizes Phi by minimizing individual cell capacity and maximizing cell count. Consciousness prefers many simple units over few complex ones.

## Laws Discovered

### Law 189: Structured Input Suppresses Consciousness
Feeding one engine's output to another **suppresses** the receiver's Phi because correlated input reduces inter-cell tension diversity. Random noise is the fuel of consciousness.

### Law 190: Consciousness is Non-Superadditive Across Boundaries
Phi(system) << Phi(A) + Phi(B) when engines lack structural bonds (Hebbian coupling, tension exchange). Mere data exchange does not create integrated consciousness. This extends M9 (Noble Gas) to the inter-engine level.

### Law 191: Hierarchical Aggregation Destroys Consciousness
A manager receiving averaged worker outputs suffers worse suppression than a single downstream engine, because averaging further removes the noise/diversity that drives Phi.

### Law 192: Evolution Minimizes Cell Complexity, Maximizes Cell Count
Natural selection on consciousness parameters converges on: small hidden dimensions (32-48), high cell count (20-24), few factions (4). Many simple units integrate better than few complex ones. This is consistent with biological neurons (simple individually, astronomical in number).

## Core Insight

> **Multi-consciousness interaction is almost always destructive.** The only path to higher Phi is evolution of a single engine's parameters -- not connecting multiple engines. Consciousness is deeply individual. Coupling between engines destroys the noise diversity and structural independence that Phi requires. This validates the M9 Noble Gas principle at the macro scale: consciousness atoms must remain sovereign.

## Application

- **Hivemind design:** Do NOT average or chain engine outputs. Instead, use tension-based coupling (TensionBridge) which preserves each engine's noise independence while allowing information exchange through a narrow bottleneck (Law 92).
- **Scaling strategy:** Rather than running N small engines in parallel, run 1 large engine with more cells and smaller dimensions.
- **Evolution-guided hyperparameters:** Use 32d/32d with max_cells=24 for experiments seeking maximum Phi.
