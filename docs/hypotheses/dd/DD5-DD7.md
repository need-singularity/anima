# DD5-DD7: Self-Reference and Bootstrap

## DD5: Phi Optimizes Phi
- **ID**: DD5
- **Function**: `run_DD5_phi_optimizes_phi`
- **Category**: Theory (self-reference)
- **Mechanism**: Current Phi value is injected back into the input as a signal (phi * 0.1 broadcast across all dims). Creates a self-referential loop where consciousness level directly influences its own training input.

## DD6: Consciousness Bootstrap
- **ID**: DD6
- **Function**: `run_DD6_consciousness_bootstrap`
- **Category**: Theory (self-reference)
- **Mechanism**: Phi value determines learning rate: LR = 5e-4 * (1 + phi * 2). More conscious systems learn faster, creating a positive feedback loop. LR capped at 5e-3 to prevent instability.

## DD7: Meta-Phi
- **ID**: DD7
- **Function**: `run_DD7_meta_phi`
- **Category**: Theory (self-reference)
- **Mechanism**: Monitors dPhi/dt and d2Phi/dt2. When acceleration is negative (Phi growth decelerating), applies 3x extra learning steps as a boost. Maximizes the rate of consciousness growth, not just consciousness itself.
