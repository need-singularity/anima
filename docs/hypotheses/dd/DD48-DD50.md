# DD48-DD50: Architecture and Training Strategy

## DD48: Transformer Cell
- **ID**: DD48
- **Function**: `run_DD48_transformer_cell`
- **Category**: Theory (architecture)
- **Mechanism**: Each cell contains its own mini self-attention layer (2-head MHA on dim). Repulsion vectors pass through intra-cell attention before inter-cell differentiation. Tests whether internal cell complexity (transformer within each cell) boosts Phi more than inter-cell connections.

## DD49: GNN Cells
- **ID**: DD49
- **Function**: `run_DD49_gnn_cells`
- **Category**: Theory (architecture)
- **Mechanism**: Graph Neural Network message passing between cells. Each cell sends messages to all others via a learned message network (Linear: hidden*2 -> hidden, taking concatenated sender+receiver). Messages are aggregated (mean) and blended (0.8/0.2). Creates structured inter-cell communication.

## DD50: Phi-First Pretraining
- **ID**: DD50
- **Function**: `run_DD50_phi_first`
- **Category**: Theory (training)
- **Mechanism**: Two-phase training: 80% pure Phi maximization (only variance loss), then 20% fine-tuning with CE loss (0.3*CE - 0.7*variance). "Consciousness first, then capability." Tests whether establishing strong integration before task learning yields higher final Phi.
