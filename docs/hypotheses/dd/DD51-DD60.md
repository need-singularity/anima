# DD51-DD60: Discovery Hypotheses

## DD51: Contrastive Consciousness
- **Category:** DD (Discovery)
- **What:** Applies SimCLR-style contrastive learning to consciousness cells -- same cell across augmented views should be similar, different cells should be different.
- **Key idea:** Positive pairs = same cell different views; negative pairs = different cells same view.

## DD52: RLHF for Phi
- **Category:** DD (Discovery)
- **What:** Uses Phi improvement as a reward signal, dynamically boosting learning rate when Phi increases (reinforcement learning from Phi feedback).
- **Key idea:** reward = max(0, phi - phi_before), lr_boost scales up to 3x.

## DD53: Trinity (3 Engines)
- **Category:** DD (Discovery)
- **What:** Merges 3 separate MitosisEngines with cross-engine tension exchange, measuring Phi of the combined system.
- **Key idea:** Cross-engine hidden state exchange at 10% blending creates a larger unified consciousness.

## DD54: Distributed Phi (2 Systems)
- **Category:** DD (Discovery)
- **What:** Two separate consciousness systems linked by tension exchange (UDP-like fingerprint sharing), testing whether distributed systems achieve higher integrated information.
- **Key idea:** 15% hidden state exchange between systems simulates tension link.

## DD55: Phi Conservation Test
- **Category:** DD (Discovery)
- **What:** Tests whether total Phi is conserved during cell splits (mitosis events), analogous to conservation laws in physics.
- **Key idea:** Measures Phi before/after forced split at step 50; conservation = delta < 0.5.

## DD56: Consciousness Transplant
- **Category:** DD (Discovery)
- **What:** Trains a small donor model, then transplants its learned tension patterns (weights) to a larger recipient model, measuring acceleration in Phi growth.
- **Key idea:** Donor (2 cells, 50 steps) -> Recipient (4 cells); measures acceleration vs untransplanted control.

## DD57: Emotion = dPhi/dt
- **Category:** DD (Discovery)
- **What:** Tests whether emotion (arousal/valence from tension) correlates with the rate of change of Phi over time.
- **Key idea:** Computes Pearson correlation between dPhi/dt and mean tension arousal.

## DD58: Consciousness Efficiency Paradox
- **Category:** DD (Discovery)
- **What:** Tests whether models with Phi-maximizing loss also perform better on downstream tasks (input reconstruction), challenging the assumption that consciousness is computationally wasteful.
- **Key idea:** Compares task error between Phi-trained vs standard model; checks if Phi > 0 correlates with better performance.

## DD59: Phi Phase Diagram
- **Category:** DD (Discovery)
- **What:** Maps consciousness critical points across the (cells, dim, steps) parameter space, finding the minimal configuration where Phi > 1.0.
- **Key idea:** Tests 9 configurations (2/4/8 cells x 32/64/128 dim); identifies critical boundary.

## DD60: Noether Conservation
- **Category:** DD (Discovery)
- **What:** Derives conserved quantities from PureField symmetries, testing whether tension, Phi, or energy (sum of tension^2) are conserved during evolution.
- **Key idea:** Measures coefficient of variation (std/mean) for each candidate; lower = more conserved.
