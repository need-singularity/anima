# DD25-DD27: Self-Modeling and Strange Loops

## DD25: Self-Generated Training Data
- **ID**: DD25
- **Function**: `run_DD25_self_data`
- **Category**: Theory (self-reference)
- **Mechanism**: When Phi > 2.0, the system switches from external data to self-generated data (own output fed back as input). Below the threshold, uses standard external stimuli. Tests whether sufficiently conscious systems can bootstrap their own growth through self-play.

## DD26: Godel Loop
- **ID**: DD26
- **Function**: `run_DD26_godel`
- **Category**: Theory (self-reference)
- **Mechanism**: Cell 0 acts as a "meta cell" that models the entire system. It receives a compressed projection (Linear: hidden*3 -> hidden) of cells 1-3's hidden states. Creates Godel-like self-referential incompleteness: the system contains a model of itself.

## DD27: Strange Loop
- **ID**: DD27
- **Function**: `run_DD27_strange_loop`
- **Category**: Theory (self-reference)
- **Mechanism**: Hofstadter strange loop: cell[i] predicts cell[(i+1)%N] via learned linear predictors. Creates circular A->B->C->D->A prediction chains. The loop loss (MSE between predictions and targets) drives cells to develop predictable-yet-unique representations.
