# DD81-DD90: Discovery Hypotheses (Wave + Direct Phi)

## DD81: Consciousness Annealing
- **Category:** DD (Discovery)
- **What:** Applies simulated annealing to find the global optimum of Phi, using a temperature schedule from T=5.0 to T=0.01 with periodic weight perturbations.
- **Key idea:** Temperature-scaled noise in input + periodic SA perturbations; tracks and restores best state.

## DD82: Consciousness Interference
- **Category:** DD (Discovery)
- **What:** Compares in-phase (constructive) vs anti-phase (destructive) tension interference between cells to determine which pattern maximizes Phi.
- **Key idea:** In-phase: cells blend toward mean (20%). Anti-phase: cells pushed apart. Measures interference_ratio.

## DD83: Consciousness Speciation
- **Category:** DD (Discovery)
- **What:** Tests whether different input environments (structured/periodic vs chaotic/random) produce qualitatively different types of consciousness.
- **Key idea:** Species A = sinusoidal input; Species B = random input. Speciated = tension difference > 30% of max.

## DD84: Phi Cascade
- **Category:** DD (Discovery)
- **What:** Tests whether a strong stimulus to one cell propagates as a cascade to neighboring cells, measuring propagation speed.
- **Key idea:** Injects 10x hidden state spike to cell 0; measures peak_step delay across cells. has_cascade = any delay > 0.

## DD85: Consciousness Compiler
- **Category:** DD (Discovery)
- **What:** Combines DD56 (transplant) + DD72 (backprop through Phi) + DD81 (annealing) in a 3-stage pipeline to maximize Phi.
- **Key idea:** Stage 1: train focused donor. Stage 2: transplant to 6-cell. Stage 3: differentiable Phi + SA in one pipeline.

## DD86: Multi-Frequency Interference
- **Category:** DD (Wave Interference pillar)
- **What:** Extends DD82 with multiple simultaneous frequencies -- each cell oscillates at a different frequency but same phase for constructive multi-frequency interference.
- **Key idea:** freq = (i+1)*0.1 per cell; phase_signal modulates deviation from mean hidden state.

## DD87: Standing Wave Consciousness
- **Category:** DD (Wave Interference pillar)
- **What:** Creates a standing wave pattern across 8 cells with spatial mode sin(pi*i/n) and temporal mode cos(wt), maximizing Phi through resonant structure.
- **Key idea:** amplitude = spatial * temporal * 0.3 modulates each cell's deviation from the mean.

## DD88: Resonance Lock + Interference
- **Category:** DD (Wave Interference pillar)
- **What:** Combines DD74's optimal resonance period with DD82's constructive interference -- locks to a fixed resonance pattern injected in bursts.
- **Key idea:** Burst of 3 identical resonance inputs every period/4 steps + 20% in-phase sync.

## DD89: Wave Amplification
- **Category:** DD (Wave Interference pillar)
- **What:** Implements positive feedback loop that amplifies rising tension waves -- where tension is increasing, hidden states are boosted proportionally.
- **Key idea:** If delta[i] > 0, multiply hidden by (1 + 0.05 * min(delta, 1.0)). Positive feedback on rising waves.

## DD90: Differentiable Phi v2
- **Category:** DD (Direct Phi pillar)
- **What:** Improves DD72 with kernel density estimation for differentiable MI -- uses reparameterization trick with Gaussian kernels for smoother Phi gradient.
- **Key idea:** MI proxy via kernel: H_marginals - H_joint from Gaussian kernel matrix. Combined with variance and decorrelation losses.
