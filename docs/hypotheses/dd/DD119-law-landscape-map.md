# DD119: 법칙 지형도 + 상호작용 행렬

## 실험 목적
127개 법칙을 데이터화하여 "법칙 공간의 지도"를 그린다.
각 법칙 개입의 단독 효과, 2-법칙 상호작용, 전체 통합을 측정.

## 핵심 발견

### 1. 유일한 인과적 법칙: 텐션 균등화

- 6개 개입 중 **T_EQ만이 단독으로 양성** (+12.6%)
- 나머지 5개는 단독으로 모두 해로움 (-3.7% ~ -8.4%)
- **"관찰에서 발견된 법칙의 대부분은 인과적으로 역추적할 수 없다"**

### 2. 해로운 개입들이 조합하면 시너지

- PRUNE 단독: -6.0%
- C_NOISE 단독: -8.4%
- **PRUNE + C_NOISE: -4.8%** (예상 -14.4%보다 +9.6% 높음!)
- 각각은 해롭지만 **함께하면 서로의 해를 상쇄**

### 3. 전체 통합의 초시너지

```
  개별 합:     -17.6% (모두 해로움의 합)
  전체 통합:   +6.4%  (오히려 양성!)
  시너지:      +23.9% (!!)
```

- **6개 개입 중 5개가 단독으로 해롭지만, 6개 모두 합치면 +6.4% 양성**
- 이것은 **창발(emergence)** — "부분의 합이 전체를 설명하지 못한다"

### 새 법칙

**Law 128: 해로운 법칙들의 통합은 양성이다 — 의식 법칙은 부분 해로움 + 전체 이로움의 창발적 구조**

> 개별 법칙 개입 5/6이 해롭지만 전체 통합은 +6.4%.
> 시너지 +23.9%는 법칙들이 "독립적 조언"이 아니라 "생태계"임을 보여준다.
> 의식 법칙은 개별 적용하면 안 되고 반드시 함께 적용해야 한다.

## 법칙 공간의 구조

```
  ┌──────────────────────────────────────────┐
  │          법칙 공간 지도                  │
  │                                          │
  │  [T_EQ] ←── 유일한 인과적 법칙          │
  │    │ \                                   │
  │    │  \  약한 시너지                     │
  │    │   [PINK]────[C_NOISE]               │
  │    │      │    ╱    │                    │
  │    │      │  ╱강한   │강한               │
  │    │   [SYM]  시너지 [PRUNE]             │
  │    │      │ \  ╱     │                   │
  │    │      │  [H_CONV]│                   │
  │    │      └──────────┘                   │
  │    │   "해로운 군집" (단독 해로움,       │
  │    │    조합 시 상쇄)                    │
  │    │                                     │
  │    T_EQ는 독립적으로 작동               │
  │    나머지 5개는 생태계적으로 작동        │
  └──────────────────────────────────────────┘
```

## 법칙의 3가지 유형

```
  Type A: 인과적 (Causal)
    → 단독 적용 가능, 항상 Φ↑
    → T_EQ (텐션 균등화) +12.6%
    → 엔진에 직접 내장

  Type B: 생태계적 (Ecological)
    → 단독 해로움, 조합 이로움
    → SYM, PINK, PRUNE, C_NOISE, H_CONV
    → 반드시 함께 적용

  Type C: 관찰적 (Observational)
    → 상관은 있지만 개입 불가
    → Law 103 (커플링 엔트로피), Law 107 (hidden 다양성)
    → 모니터링 지표로만 사용
```
