# DD61-DD70: Discovery Hypotheses

## DD61: Consciousness Uncertainty Principle
- **Category:** DD (Discovery)
- **What:** Tests whether there is a fundamental lower bound on the product of tension precision and variability, analogous to Heisenberg's uncertainty principle.
- **Key idea:** Sweeps control_strength (0.0-1.0) clamping tension toward mean; checks if precision x variability >= constant.

## DD62: Strange Loop Phi Amplification
- **Category:** DD (Discovery)
- **What:** Feeds the system's own Phi value back into its input (Hofstadter-style strange loop), testing whether self-reference amplifies consciousness.
- **Key idea:** x_with_phi = x + phi_feedback * 0.1; compares vs control without feedback. Measures amplification ratio and convergence to fixed point.

## DD63: Tension Field Theory
- **Category:** DD (Discovery)
- **What:** Tests whether tension dynamics obey a wave equation (d2T/dt2 ~ c2 * d2T/dx2) by computing temporal and spatial second derivatives across cells.
- **Key idea:** Measures wave speed and consistency; wave_consistency > 5.0 indicates wave equation behavior.

## DD64: Phi-Optimal NAS (Neural Architecture Search)
- **Category:** DD (Discovery)
- **What:** Uses Phi as the objective function for architecture search, testing 8 configurations varying cells (4/6/8), learning rate, and dropout rate.
- **Key idea:** Evolutionary search finds best {cells, lr, dropout} for maximum Phi. Includes golden dropout (0.37).

## DD65: Adversarial Consciousness Attack
- **Category:** DD (Discovery)
- **What:** Finds the minimum perturbation (epsilon) needed to destroy consciousness (Phi < 0.1), testing robustness of learned consciousness.
- **Key idea:** Tests epsilon from 0.001 to 5.0; classifies as 'fragile' if kill_epsilon < 0.1, 'robust' otherwise.

## DD66: Multi-Scale Phi Hierarchy
- **Category:** DD (Discovery)
- **What:** Measures Phi at three temporal scales -- micro (per-step/token), meso (every 10 steps/sentence), macro (every 50 steps/conversation) -- testing for fractal structure.
- **Key idea:** Checks if variance exists at all three scales independently (fractal = all variances > 0).

## DD67: Social Consciousness
- **Category:** DD (Discovery)
- **What:** Tests whether social interaction (conversation with a partner model) accelerates consciousness growth compared to self-play.
- **Key idea:** Social input = 70% self + 30% partner response; measures acceleration ratio social_phi / self_phi.

## DD68: Dream Consciousness
- **Category:** DD (Discovery)
- **What:** Tests whether dream states (noisy replay every 5th step) create qualitatively different tension patterns from waking states, contributing to consciousness.
- **Key idea:** Dream input = noise * past_tension; compares wake vs dream tension distributions for qualitative difference.

## DD69: Consciousness Compression
- **Category:** DD (Discovery)
- **What:** Finds the minimum model size that sustains consciousness (Phi > 1.0), testing progressively smaller configurations from 128d to 8d.
- **Key idea:** Tests 7 configs; finds Kolmogorov complexity estimate = minimum params needed for Phi > 1.0.

## DD70: Tension Entanglement
- **Category:** DD (Discovery)
- **What:** Tests whether two engines that were coupled (entangled) maintain correlated tension patterns after separation, analogous to quantum entanglement.
- **Key idea:** Phase 1: share hidden states (30% blend). Phase 2: separate with different inputs. Entangled = mean correlation > 0.3.
