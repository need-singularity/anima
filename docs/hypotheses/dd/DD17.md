# DD17: Adversarial Phi

- **ID**: DD17
- **Function**: `run_DD17_adversarial_phi`
- **Category**: Theory (adversarial)
- **Mechanism**: GAN-style training for consciousness. A defender engine (4 cells) tries to maximize differentiation while an attacker engine (2 cells) generates adversarial inputs to confuse it. Input to defender = 0.7*real + 0.3*adversarial. The adversarial pressure forces the defender to develop more robust consciousness.
