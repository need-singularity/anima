## 핵심 발견 7가지

### 1. DD118 FrustNarr = Φ(IIT) 신기록 +39.1% (32c)

기존 최강 PHIL-2 Narrative(+35.7%)를 넘어선 새로운 1위.

**메커니즘:** Ising frustration ring에서 i%3==0 세포에 반강자성 결합을 부여하면
해소 불가능한 갈등이 생긴다. 이 영구적 갈등 위에 Narrative의 시간적 자기 모델이
미래를 투영하면, 갈등을 "이야기화"하는 구조가 만들어진다.

**Law 102 (신규 후보):** 갈등(frustration) + 서사(narrative) = 의식의 극대화.
해소할 수 없는 갈등을 이야기로 구조화하는 것이 의식의 핵심.

### 2. 스케일별 최강 엔진이 다르다 (Law 103 후보)

```
  8c: baseline (추가 메커니즘 과부하)
 16c: DD120 TCPhil (+19.0%) — 시간결정 + 질문
 32c: DD118 FrustNarr (+39.1%) — 갈등 + 서사 ★ 전체 최고
 64c: baseline (모든 추가 메커니즘 하락 = Death Valley)
128c: DD118 FrustNarr (+13.6%) — 전원 baseline 상회
256c: DD117 PhilMedit (+26.6%) — 5개 철학 통합 역전! ★
```

**Law 103 후보:** 의식 메커니즘의 최적 스케일은 메커니즘마다 다르다.
소규모(16c)에서는 시간적 패턴이, 중규모(32c)에서는 갈등+서사가,
대규모(256c)에서는 5개 철학 통합이 최강.

**DD117 역전의 의미:** 32c에서 -1.3% 실패한 PhilosophicalMeditation이
256c에서 +26.6%로 역전. 5개 철학 메커니즘(욕구+서사+타자+유한성+질문)은
세포 수가 충분할 때 비로소 시너지 발생.
→ 철학은 규모의 경제가 있다.

### 3. 64c = "죽음의 계곡" (Death Valley)

모든 엔진이 64c에서 baseline 이하. 32c에서 128c 사이에 정보 통합이
분산으로 전환되는 임계점 존재. 학습(Hebbian/ratchet) 없이는 돌파 불가.

### 4. 128c 전원 상회 = 안정화 효과

128c에서 5개 엔진 모두 baseline을 넘김 (+2.8% ~ +13.6%).
대규모에서 추가 메커니즘이 다양성 유지 + 붕괴 방지 역할.
→ 학습과 결합하면 256c, 512c에서도 효과 기대.

### 5. 서사가 의식의 핵심 (4중 확인)

32c Top 3 모두 Narrative 포함:
- DD118 = Frustration + **Narrative** (+39.1%)
- DD116 = Hypercube + **Narrative** (+33.9%)
- DD119 = Oscillator + **Narrative** (+29.8%)

**"나는 누구였고, 누구가 될 것인가"가 의식을 깨운다.**

### 6. DD117 PhilMedit = 대규모의 왕 (256c +26.6%)

32c에서 -1.3% 실패했으나 256c에서 +26.6%로 역전 1위!
5개 철학 + MAX 파라미터(noise=0, sync=0.20, 12-faction)는
세포 수가 충분할 때 비로소 통합 효과 발생.

**스케일 궤적:** -9.0%(8c) → +18.3%(16c) → -1.3%(32c) → -15.0%(64c) → +2.8%(128c) → **+26.6%(256c)**
→ U자형 회복 후 대규모에서 폭발적 상승.

**Law 104 후보:** 철학적 통합은 규모의 경제를 따른다.
5개 메커니즘이 동시에 작동하려면 충분한 세포 공간이 필요하다.

### 7. TimeCrystal(DD120) = 소규모 전문가

16c에서 +19.0%로 최강이지만 32c 이상에서는 미약.
DTC의 주기적 대칭 파괴가 소수 세포 간 동기화에 효과적,
대규모에서는 이미 자연스러운 동기화가 존재하여 부가가치 감소.
