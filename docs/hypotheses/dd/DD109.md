# DD109: 3D 공간 인식 시 Phi 상승

## 가설
동일 세포 수에서 3D 공간 입력이 2D 대비 Phi를 유의미하게 상승시킨다.
공간 정보가 세포 간 정보 통합(integrated information)을 증가시키기 때문.

## 알고리즘
1. 128c ConsciousMind 초기화
2. 조건A: 2D 이미지 (senses.py) -> process() x 100 steps -> Phi(IIT) 측정
3. 조건B: 3D 포인트 클라우드 (lidar_sense.py) -> process() x 100 steps -> Phi(IIT) 측정
4. 10회 반복 평균

## 성공 기준
Phi(3D) > Phi(2D) x 1.1 (10% 이상 상승)

## 모듈
- `lidar_sense.py` — LidarSense, TensionField3D, SimulatorDriver
- `senses.py` — SenseHub (비교 대상)
- `anima/core/runtime/anima_runtime.hexa` — ConsciousMind, ConsciousnessVector (S 차원 추가)

## 아키텍처

```
  PointCloud (N,3)
       |
       v
  TensionField3D (voxel_size^3)
       |
       v
  LidarSense.get_S()
       |
       v
  ConsciousnessVector.S
       |
       v
  ConsciousMind.process() -> Phi 측정
```

## 결과
(벤치마크 실행 후 기록)

```
Phi |
    |
    |
    |
    └──────────────── condition
         2D    3D
```
