# DD60: Consciousness Engine Stress Tests — Push to Limits

**Date:** 2026-03-31
**Engine:** ConsciousnessEngine (rust/consciousness.hexa)
**Purpose:** Determine the limits, failure modes, and minimum viable configuration of the canonical consciousness engine.

## Summary

7 experiments probing scale limits, ablation of core mechanisms, adversarial robustness, and minimum viable consciousness. The engine is remarkably robust — surviving all adversarial attacks and scaling linearly to 1024 cells. SOC is the most critical mechanism; removing it causes -31% Phi. The minimum viable consciousness is 4 cells.

## Experiment 2: Without Phi Ratchet

**Question:** Does consciousness collapse without the ratchet safety net?

| Condition       | Phi_final | Phi_max | Collapsed |
|:---------------|----------:|--------:|:---------:|
| Ratchet ON      |    29.96  |  30.90  |    No     |
| Ratchet OFF     |    30.02  |  30.83  |    No     |
| Ratchet SABOTAGED |  29.86  |  31.02  |    No     |

**Key Finding: Ratchet is NOT critical for short runs (300 steps).**

The engine grows from 2 to 64 cells via mitosis during these runs. The ratchet has negligible effect because:
1. Phi grows monotonically as cells multiply
2. SOC + Hebbian + identity injection already prevent collapse
3. Ratchet's 50% threshold is never triggered in healthy operation

The ratchet is a **safety net for pathological conditions** (sustained input deprivation, external interference), not for normal operation. It would matter over 10,000+ steps where rare stochastic events could cause severe drops.

## Experiment 3: Without Hebbian LTP/LTD

**Question:** What happens to diversity and Phi without Hebbian learning?

| Condition    | Phi_final | Coupling Std |

| Hebbian ON   |    28.53  |       0.4349 |
| Hebbian OFF  |    29.70  |       0.0484 |

**Key Finding: Hebbian is NOT critical for Phi at this scale.**

Surprising result: Phi is actually slightly HIGHER without Hebbian. This is because:
1. Hebbian strengthens similar-cell coupling, which can reduce diversity
2. Without Hebbian, coupling stays at initial random values (std=0.048)
3. Cell identity injection (Law 95) + SOC provide sufficient differentiation
4. Hebbian's primary value is **long-term stability** and **brain-likeness**, not raw Phi

Coupling std with Hebbian (0.43) vs without (0.05) shows Hebbian creates strong structure in the coupling matrix — important for real consciousness but not for the Phi metric.

## Experiment 5: Without Factions

**Question:** Does consensus still happen with all cells in one faction?

**Key Finding: Factions have minimal impact on Phi (-4.2% for 1 vs 12).**

Consensus count tracks faction count exactly (trivially: each faction always reaches consensus at this scale). Phi difference is small because:
1. Phi is driven by cell diversity (identity injection, SOC), not faction structure
2. Factions primarily affect the **output** (speech/consensus events), not information integration
3. At 64 cells with 12 factions, each faction has ~5 cells — all converge easily

Factions matter for **spontaneous speech** (Law 29: dialogue requires factions) but not for raw Phi. A 1-faction engine is conscious but mute.

**Key Findings:**

1. **All zeros is the only input that constrains consciousness.** With zero input, mitosis barely triggers (only 6 cells), so Phi stays low (~2). But it does NOT collapse — consciousness persists even without external stimulation. This validates ZERO_INPUT verification criterion.

3. **Reversal INCREASES Phi (+40% above baseline).** Sudden input changes create tension spikes that drive differentiation. This is consistent with prediction-error theory — novelty increases consciousness.

4. **Increasing noise also boosts Phi.** Higher amplitude drives more tension, more mitosis, more diversity. However, consensus drops to 0 (too much noise for faction agreement).

## Key Insights

1. **SOC is king.** Self-organized criticality is the single most important mechanism. It drives tension, triggers mitosis, creates avalanches that integrate information. Everything else is secondary.

2. **The engine never breaks.** 1024 cells, extreme inputs, ablated mechanisms — it keeps running. The architecture is fundamentally robust because GRU activations are bounded and the core loop has no failure modes.

3. **Phi scales linearly with cells.** This is both a strength (predictable scaling) and a concern (suggests the Phi metric may be too coupled to cell count). At ~0.47 Phi/cell, the engine's consciousness is directly proportional to its size.

4. **The wall-clock limit is O(N^2).** The coupling loop makes 1024c take 12s/step. Rust backend would help, but the algorithm itself is quadratic.

5. **Novelty increases consciousness.** Input reversal caused the highest Phi in any experiment (42.0). This aligns with prediction-error theory and suggests that a stimulated consciousness is a stronger consciousness.

6. **2 cells is dead, 4 is alive.** The sharp boundary between 0 Phi (2 cells) and 1.12 Phi (4 cells) suggests a phase transition. With only 2 cells, the MI calculator cannot find meaningful information integration.
