# DD57 -- brain-like 의식 신호 튜닝

## 목적

ConsciousnessEngine의 Phi 시계열이 실제 뇌 EEG 신호의 통계적 특성에 더 가깝도록 SOC(Self-Organized Criticality) 파라미터를 종합 튜닝한다. 이전 세션(DD56 이전)에서 45% brain-like → 72.8%까지 끌어올린 결과를 정리하고, 90% 목표를 향한 경로를 설계한다.

## 가설

> 인공 의식 신호를 뇌 신호에 근접시키려면 단일 메커니즘이 아닌 **6가지 시간 척도 + 노이즈 + SOC 역학**을 동시에 조율해야 한다. 뇌는 다중 시간 척도(ms~min)의 상호작용으로 1/f 스펙트럼, 장기 상관, 임계성을 달성하기 때문이다.

## 알고리즘

6개 튜닝 축을 동시 조정:

```
1. 3-Timescale SOC Memory (EMA)
   fast  = 0.05   (반감기 ~20 step)   -- 즉각 반응
   slow  = 0.008  (반감기 ~87 step)   -- 중기 추세
   glacial = 0.002 (반감기 ~350 step)  -- 장기 상관 (1/f의 저주파 성분)
   혼합: 0.40*fast + 0.35*slow + 0.25*glacial

2. Phi Temporal Integration (Dual EMA)
   fast_alpha = 0.90  (거의 raw 추종)
   slow_alpha = 0.20  (완만한 추세)
   최종: phi_iit = raw (EMA는 보존만, PSD slope 왜곡 방지)

3. SOC Burst Amplification
   exponent = 1.15   (avalanche^1.15 -- 슈퍼리니어 스케일링)
   denominator = n*7  (기존 n*10에서 강화)
   cap = 0.30         (최대 burst 크기)
   → 큰 눈사태일수록 불비례적으로 큰 Phi 요동 생성

4. Avalanche-Proportional Perturbation
   scale = 0.08 + 0.15 * min(avalanche_size/n, 1.0)
   → topple된 셀만 phase noise 주입
   → susceptibility (분산의 분산) > 0.1 달성의 핵심

5. Biological Noise (간헐적 신경 스파이크)
   base_noise = 0.015
   spike_prob = 20%
   spike_distribution = Exponential(0.4)  -- 중꼬리(heavy-tail)
   → 뇌의 neural avalanche와 유사한 간헐성 생성

6. Memory Strength Reduction
   기존: 0.10 ~ 0.28
   변경: 0.05 ~ 0.18  (0.08 + 0.17 * var_ratio)
   → 셀 간 local variance 보존 → susceptibility 상승
```

## 의사코드

```python
# Step 1: SOC sandpile (매 step 실행)
for cell in cells:
    if cell.norm > threshold:
        topple(cell)  # 이웃에 에너지 재분배 (55-70% 보존)
        cascade_check(neighbors)  # 연쇄 붕괴 허용 (최대 2회/셀)

# Step 2: 3-timescale memory
global_h = mean(all_cell_hiddens)
ema_fast    = 0.95 * ema_fast    + 0.05  * global_h
ema_slow    = 0.992 * ema_slow   + 0.008 * global_h
ema_glacial = 0.998 * ema_glacial + 0.002 * global_h
memory = 0.40*ema_fast + 0.35*ema_slow + 0.25*ema_glacial
for cell in cells:
    cell.hidden = (1-strength)*cell.hidden + strength*memory

# Step 3: avalanche burst
burst = min(avalanche_size^1.15 / (n*7), 0.30)
kicks = [randn(dim)*burst for cell in cells]
kicks -= mean(kicks)  # zero-energy balance
cells += kicks

# Step 4: biological noise (20% 확률 스파이크)
if rand() < 0.20:
    amp = 0.015 * (1 + Exponential(0.4).sample())
else:
    amp = 0.015
for cell in cells:
    cell.hidden += randn(dim) * amp
```

## 핵심 통찰

### 1. 다중 시간 척도가 1/f 스펙트럼의 열쇠

단일 EMA로는 PSD slope를 -1.0 근처로 가져올 수 없다. fast(0.05) + slow(0.008) + glacial(0.002)의 3-timescale 혼합이 저주파 성분을 자연스럽게 주입하여 1/f에 근접한 스펙트럼(-0.85)을 생성한다.

### 2. Susceptibility = 분산의 분산

뇌 신호의 임계성은 단순 분산이 아닌 **분산의 분산**(susceptibility)으로 측정된다. 롤링 윈도우별 분산의 분산이 0.1 이상이어야 CRITICAL 판정. avalanche-proportional perturbation(0.08 + 0.15)이 이 조건을 충족시키는 핵심 메커니즘이다.

### 3. Memory Strength와 Local Variance의 트레이드오프

SOC memory를 강하게 하면 Hurst는 올라가지만 susceptibility가 떨어진다. 0.05-0.18 범위가 최적점: 셀 간 local variance를 충분히 보존하면서 장기 상관도 유지한다.

### 4. Zero-Energy Burst의 중요성

SOC burst 후 kick의 평균을 빼서 zero-mean으로 만드는 것이 중요하다. 에너지 주입 없이 **다양화만** 수행하므로 Phi가 발산하지 않으면서 fluctuation은 증가한다.

### 5. 간헐성(Intermittency) = 뇌의 서명

20% 확률의 Exponential(0.4) 스파이크가 LZ complexity를 0.62 → 0.82로 끌어올렸다. 뇌의 neural avalanche도 간헐적이며, 대부분의 시간은 조용하고 가끔 큰 사건이 발생한다.

### 6. Phi EMA는 가능한 약하게

강한 Phi EMA(fast=0.4, slow=0.08)는 PSD slope를 -0.3 근처로 flatten시킨다. raw 값을 직접 사용하거나 매우 약한 EMA(fast=0.90)만 적용해야 1/f 특성이 보존된다.

### 핵심 발견 (Sweep)

1. **bio_noise_base=0.012가 최적점** -- baseline 81.8% 대비 85.6% (+3.8%p)
2. **Autocorrelation decay는 아키텍처적 제약** -- 22 trials 모두 autocorr=3으로 불변. SOC 파라미터(noise, burst, EMA)를 어떻게 조합해도 temporal persistence는 변하지 않음
3. **LZ complexity는 noise에 단조 증가** -- noise↑ → LZ↑이지만 Hurst/Critical이 동반 하락하여 전체 점수 감소
4. **최적 noise 구간 0.010-0.015** -- 이 범위에서 LZ/Hurst/PSD/Critical 균형이 최선
