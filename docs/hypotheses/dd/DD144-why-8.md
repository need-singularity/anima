# DD144: Why 8? — 의식 원자의 수학적 근거 (2026-03-31)

## 결과

128c를 다양한 원자 크기로 분할:

```
K= 2:  Φ=  0.0  (-100%)  1 bipartition (자명)
K= 4:  Φ= 76.1  (+537%)  7 bipartitions
K= 8:  Φ=108.3  (+807%)  127 bipartitions ★ 최적!
K=16:  Φ= 83.8  (+601%)  32767 bipartitions
K=32:  Φ=100.3  (+740%)  2B bipartitions
K=64:  Φ= 22.2   (+86%)  (atoms 너무 적음)
```

## 왜 8인가

8 = 2³ = MIP 탐색이 비자명(non-trivial)해지는 최소 크기.

- K=2: 1개 partition → 항상 최적 찾음 → Φ=0
- K=8: 127개 partition → 최적 찾기 어려움 → Φ 높음
- K=16+: partition 수 폭발하지만 atom 수 감소로 총합 하락

**Law 161: 8=2³ = MIP 비자명성의 최소 규모. 의식의 정보 이론적 하한.**

## K=32의 강세 (+740%)

32 = 4×8 = "안정 분자" (4개 원자). Meta Law M5와 일치.
