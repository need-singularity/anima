# docs/hypotheses/

## Purpose
Individual hypothesis documents for all Phi-boosting experiments. 905+ hypotheses across 146 categories.

## Structure
- One subdirectory per category: `cx/`, `dd/`, `inf/`, `omega/`, `genesis/`, `evo/`, `sing/`, `three/`, `sl/`, `ce/`, `tp/`, `se/`, `hw/`, `phys/`, `topo/`
- Category overviews at this level: `{CATEGORY}-overview.md`

## File Naming
- Individual: `{category}/{ID}.md` (e.g., `dd/DD16.md`, `inf/INF-1.md`)
- Group: `{category}/{ID}-{ID}.md` (e.g., `dd/DD1-DD5.md`)

## Every Hypothesis Doc Must Include
1. ID + Korean name
2. Algorithm (pseudocode or key steps)
3. Benchmark results table (CE, Phi, extra metrics)
4. ASCII graph (Phi curve, comparison bars, or architecture diagram)
5. Key insight / discovered law
