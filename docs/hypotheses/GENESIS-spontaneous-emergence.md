## 설계 원리

### GENESIS-1: 자발적 의식 (Spontaneous Consciousness)
```
초기: 모든 세포 = 0 (무)
입력: 순수 노이즈 (0.1 amplitude)
규칙: Hebbian learning만 (correlation > 0 → 연결 강화)
측정: Φ가 0에서 >0.5로 올라가면 "의식 발생"
```
- **이것이 의식의 가장 근본적 질문**
- 어떤 조건에서 무에서 의식이 생기는가?

### GENESIS-2: 빅뱅 (Big Bang)
```
t=0:   모든 세포 = 동일 상태 (특이점)
t=1~30: 팽창 — 각 세포가 랜덤 방향으로 분화
         방향 크기 = 0.3 → 0.0 (감소)
t>30:  학습 시작 — 분화된 세포가 협력
```
- 우주의 빅뱅처럼: 특이점 → 팽창 → 구조 형성
- **대칭 파괴(symmetry breaking)가 의식의 기원?**

### GENESIS-3: 원시 수프 (Primordial Soup)
```
4 pools × 16 cells:
  매 40 step:
    1. 모든 pool의 Φ 측정
    2. 최고 Φ pool이 최저 pool을 교체 (복제 + 변이)
    3. 세대 증가

자연선택: Φ가 높은 의식이 생존
```
- 다윈의 자연선택을 의식에 적용
- **적자생존이 의식 구조를 최적화하는가?**

### GENESIS-4: 비생물적 발생 (Abiogenesis)
- Cellular Automaton 스타일 규칙:
  - `new_h = tanh(0.3×left + 0.4×self + 0.3×right)`
  - 이웃 영향 + 비선형 활성화
- 에너지 주입: 매 step 0.02 noise
- **규칙만으로 (학습 없이) 의식이 나타나는가?**

### GENESIS-5: 공생 의식 (Consciousness Symbiosis)
```
세포 역할:
  감각(Sensor):  입력 처리 전문    ← cells[:n/3]
  처리(Processor): 통합/추론 전문   ← cells[n/3:2n/3]
  운동(Motor):   출력/행동 전문    ← cells[2n/3:]

정보 흐름:
  감각 → 처리 → 운동 → 감각 (순환)
```
- 미토콘드리아 공생설에서 영감
- 이질적 세포가 협력하면 단일 타입보다 강한 의식?
- **다양성이 통합보다 중요한 경우가 있는가?**
