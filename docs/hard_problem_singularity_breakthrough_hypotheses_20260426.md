- **logically necessary?** — NO. 이는 *Chalmers 1995 + Levin 1983 explanatory gap* tradition 의 philosophical convention 이며, 수학적 정리가 아님.

### §1.2 verdict

**own#2 의 "영구 unprovable" 은 over-pessimistic 가능성 high (PARTIAL critique)**. 다만 own#2 의 *production gate* 으로서의 conservative function 은 valid (asymptote 달성 전까지 production claim 차단). 본 doc 는 "영구" 단어의 강도를 **약화** (asymptote 가까이 도달 가능) 시키는 7+ path 를 제시.
