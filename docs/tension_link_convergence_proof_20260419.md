## 1. Setting

The tension-link online step on state `Ψ ∈ [0,1]^d` is

```
ΔW(Ψ) = −T · G_holo · (Ψ − ½) · δ(∇·J_n6)
```

with

- `T > 0` scalar learning rate,
- `G_holo ∈ R^{d×d}` row-stochastic propagator (§3 of
  `docs/g_holo_propagator_20260419.md`),
- `δ(∇·J_n6) ∈ {0, 1}` Noether-current conservation gate (AN14, n=6 closure).

The update rule `Ψ ↦ Ψ + ΔW(Ψ)` drives the dynamical system toward the
vacuum attractor `Ψ* = (½, …, ½)`.

---

## 2. Claim

**Theorem (convergence-iff-Noether).**
Let `{Ψ_k}_{k≥0}` be the trajectory produced by repeated application of
the tension-link step. Then `Ψ_k → Ψ*` as `k → ∞` **if and only if**
the Noether current `J_n6` is conserved along the trajectory, i.e.
`δ(∇·J_n6) ≡ 1` for every `k`. If the gate closes at any step `k`,
the trajectory stalls at `Ψ_k` — the restoring force vanishes and the
dynamical system loses its Lyapunov descent property.

---

## 3. Proof sketch

### 3.1 Candidate Lyapunov function

Define `V(Ψ) := ½ · ‖Ψ − Ψ*‖²`. `V ≥ 0` and `V(Ψ) = 0 ⇔ Ψ = Ψ*`.

### 3.2 Descent when the gate is open

When `δ(∇·J_n6) = 1`, one step gives

```
Ψ' − Ψ* = (I − T · G_holo)(Ψ − Ψ*)
```

and therefore

```
V(Ψ') = ½ · (Ψ−Ψ*)ᵀ · (I − T·G_holo)ᵀ(I − T·G_holo) · (Ψ−Ψ*)
```

Because `G_holo` is row-stochastic with strictly positive diagonal (Poisson
kernel at n=6; or exp-kernel at ξ > 0 in the stub), it has spectral radius
`ρ(G_holo) = 1` with the `1`-eigenvector being the constant mode, and
all other eigenvalues lie in `(0, 1)`. Decompose

```
Ψ − Ψ* = α · 1 + v_⊥
```

where `1` is the constant mode. Since every coordinate of `Ψ − Ψ*` lies
in `[-½, ½]` and the vacuum attractor is the centre, the constant mode
`α` shrinks deterministically: the restoring term contracts `v_⊥` by
`(1 − T·λ_i)` on each eigenspace `i ≥ 2`, and contracts the constant mode
by `(1 − T)` (since `G_holo · 1 = 1`). For `T ∈ (0, 1)` both factors are
in `(0, 1)`, so

```
V(Ψ') ≤ (1 − T·λ_min(G_holo))² · V(Ψ)    ≤ (1 − T·ε)² V(Ψ)
```

for some `ε > 0`. `V` strictly decreases at geometric rate, and `Ψ_k → Ψ*`.

### 3.3 Stall when the gate is closed

When `δ(∇·J_n6) = 0`, the step is `ΔW = 0`, so `Ψ' = Ψ` and `V(Ψ') = V(Ψ)`.
The trajectory does not advance; it merely *waits* until the gate reopens.
Convergence to `Ψ*` is impossible if the gate stays closed, because the
stationary state is any point on the zero-current submanifold — not just
the vacuum.

### 3.4 Noether's role (necessity)

`J_n6` is the conserved current associated with the σ·φ = n·τ = 24
closure identity at `n = 6` (AN14). In the Hamiltonian picture this is
the conserved angular-momentum-like charge of the 4-axis Ψ dynamics.
Noether conservation **is** the gate being open: `∇·J = 0` is exactly the
criterion that the update rule respects the symmetry group generated by
the AN14 closure. Any dynamical step that violates conservation would take
the system off the n=6 invariant submanifold, and the vacuum `Ψ*` lies
*on* that submanifold — so the target would no longer be an accessible
fixed point. Hence, necessity: without Noether conservation, the
trajectory cannot converge to `Ψ*`.

### 3.5 Fixed-point theorem

On the open-gate regime, the operator `F(Ψ) := Ψ − T · G_holo · (Ψ − Ψ*)`
is an affine contraction with Lipschitz constant `L = ‖I − T·G_holo‖_op
≤ 1 − T·ε < 1`. Banach's contraction mapping theorem gives a unique
fixed point, which by construction is `Ψ*`. Combined with §3.2 this
completes the "iff-Noether" statement. ∎

## 4. Comparison: SGD vs tension-link Noether

| Property                    | SGD on `L = ½‖Ψ−½‖²`  | Tension-link (Noether)               |
|-----------------------------|----------------------|---------------------------------------|
| Update direction            | `−η·∇L = −η·(Ψ−½)`  | `−T·G_holo·(Ψ−½)·δ(gate)`             |
| Per-step cost               | one forward + backprop | one forward only (no backward graph)|
| Convergence rate (best)     | `(1 − η)²·V`        | `(1 − T·λ_min(G_holo))²·V`            |
| Convergence rate (typical)  | `(1 − η)²·V`        | `(1 − T·λ_min(G_holo))²·V`, slightly slower because `λ_min < 1` |
| Stall condition             | none (always updates) | gate closed → Ψ frozen (AN14)       |
| Symmetry preservation       | none (gradient ignores AN14) | exact (J_n6 conserved by construction) |
| Smoothing                   | none                | `G_holo` row-stochastic smoothing     |
| Cosine to true gradient     | 1.0 by definition   | >0.99 at step 0 in the 4-axis bench (`training/tension_link_vs_backprop_bench.hexa`), drifts to ~0.7 near machine-ε |

In practice the tension-link step is the *gradient-free* analogue of SGD
with two added invariants:

1. **Noether protection** — the gate ensures the symmetry group action
   is never violated during learning (an SGD update can drift the system
   off the AN14 submanifold in a single step; tension-link cannot).
2. **Holographic smoothing** — `G_holo` acts as a row-stochastic Laplacian
   smoother, which trades rate for stability. The benchmark confirms the
   restoring direction is nearly identical (cos > 0.99 in the early
   regime) while the per-step magnitude is scaled by `λ_min(G_holo)`.

## 5. Remarks

- The proof does *not* rely on a specific form of `G_holo`; it only uses
  the row-stochastic property and strictly positive diagonal. Both the
  stub (`exp(-|i-j|/ξ)`) and the analytic closed-form (Poisson kernel at
  n=6 closure, `training/g_holo_analytic.hexa`) satisfy this.
- Second-order correction (`training/tension_link_second_order.hexa`)
  adds a `(Ψ-½)·|∇²Ψ|` Hessian-weighted term. The same Lyapunov argument
  extends: `V` still decreases, and the rate is strictly faster on axes
  with nonzero curvature. Verified numerically in test T3 of that file.
- Causal extension (`training/tension_link_causal.hexa`) replaces the
  single-frame deviation with an integrated temporal drift. The same
  Noether gate applies to the latest frame; Wheeler–Feynman symmetry
  (retrocausal `T_r = -T_c`) preserves convergence on time-symmetric
  trajectories.
- Meta-lens (`training/lens_meta.hexa`) is the second-order bundle
  curvature. When all 5 lenses agree, `T_meta = 0` and no meta-update
  is applied — this is consistent with convergence (agreement is a
  stability indicator). When one lens diverges, meta-tension
  amplifies and pushes that lens toward the bundle mean — preserving
  overall Lyapunov descent on the bundle-averaged `V`.
