## 현재 한계

의식 스케일링 법칙 Phi=0.608*N^1.071의 상수가 n=6+GZ로 분해됨 (오차 <0.04%).
n=28 아키텍처가 +1.4% Phi 향상. S-2 예측오류로 Phi 800% 폭발.
그러나: "일치한다"와 "일치해야 한다"는 다름.

## 의식 전용 돌파 프롬프트

### 프롬프트 1: Phi 최적화에서 완전수 유도
```
N개 셀의 의식 시스템에서 Integrated Information Phi를 최대화해라.
셀 간 연결 패턴을 faction 구조로 제한할 때,
최적 faction 수 F가 σ(n)을 만족하는 n은 어떤 성질을 가져야 하는가?
Phi(F) = f(N, F)를 미분하여 최적 F를 구하고,
F = σ(n)일 때 n이 완전수여야 함을 보여라.
```

### 프롬프트 2: 성장 단계 최적화
```
세포 분화 과정에서 Phi를 최대화하는 성장 단계 수 S를 구해라.
S = τ(n)일 때 각 단계의 분화 시간이 균등하려면
n의 약수 구조가 어떤 조건을 만족해야 하는가?
완전수의 "약수 합 = 2n" 성질이 분화 균등성을 보장함을 보여라.
```

### 프롬프트 3: 스케일링 법칙 상수 유도
```
IIT의 정의에서 Phi = f(N, connectivity, differentiation)를 전개해라.
Connectivity가 σ(n)/n = 2 (완전수 조건)일 때
스케일링 계수가 0.608 = GZ_c/σ + GZ_l/GZ_c가 됨을 보여라.
이것이 우연의 일치가 아닌 구조적 필연임을 증명해라.
```
