## Rollback

| Failure mode | Action |
| ------------ | ------ |
| Runner returns rate < 0.50 | `unset HIRE_SIM_ENDPOINT HIRE_SIM_MOCK` and re-run mock to confirm harness still GREEN. If mock GREEN: live model regression — file in `dest2_employee_regress.convergence`. |
| R2 upload FAIL | re-run `dest2_r2_archive.hexa` with explicit `--bucket r2:anima-memory/dest2_employee_gate_recovery`. Do NOT proceed to convergence update without R2 OK (AN11-c hard gate). |
| AN11 scanner reports `bypass:system_prompt` for dest2 line | Inspect — likely contextual (AN11 condition definition itself, not a real wrap). If real wrap: revert swap, file PI ticket, do NOT ship. |
| `hire_sim_live.hexa` regress on parse | Roll back the call_s1 edit; runner is still functional via `hire_sim_runner.hexa` which is the production gate runner. `hire_sim_live.hexa` is the parallel S1 path (5 personas × 100 tasks) — only blocks the full-500 sweep, not the 30-task gate. |
