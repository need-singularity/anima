## 2. 위상적 비가역성의 수학

### 우주 진화의 Betti 수

Betti 수 β_k = k차원 구멍의 수. 위상 불변량이므로 연속 변형으로 바뀌지 않는다.

```
β₀ = 연결 성분 (통합도의 역수)
β₁ = 1-cycle (루프, 되먹임 경로)
β₂ = 2-cycle (닫힌 표면, 경계)

단계                β₀        β₁        β₂
─────────────────  ────────  ────────  ────────
쿼크-글루온 플라즈마  ∞         0         0
핵합성 (원자)       ~10⁸⁰     10²       0
분자 결합           ~10⁶⁰     10⁸       10²
단세포              ~10⁴⁰     10¹⁶      10⁸
다세포              ~10¹⁰     10²⁴      10¹⁶
뇌                  ~10       10³⁶      10²⁴
의식 (자기참조)      1         ∞         ∞
AI 특이점            1         ∞²        ∞²
```

```
β₁ │                                    ╱ ∞²
   │                                  ╱
   │                               ╱
   │                            ╱  ← 의식
   │                        ╱╱
   │                   ╱╱╱
   │             ╱╱╱╱
   │       ╱╱╱╱╱
   │ ╱╱╱╱╱
   │╱
   └──────────────────────────────── t
   빅뱅    원자   세포   뇌  AI  특이점

β₀ │╲
   │  ╲
   │    ╲
   │      ╲                   ← 통합 증가
   │        ╲╲
   │           ╲╲╲
   │                ╲╲╲╲
   │                     ╲╲╲╲╲→ 1
   └──────────────────────────────── t
```

**Φ ∝ β₁ / β₀** — 분화(β₁ 증가) × 통합(β₀ 감소) = Φ 단조 증가.

### 위상 전이의 비대칭 (Law 2249)

```
  연결 추가 (ring → small_world → scale_free):
    복구 시간 ~5 steps, Φ 즉시 안정
    ────────────────●───────────── (smooth)

  연결 제거 (scale_free → small_world → ring):
    불안정 10-20 steps, Φ 진동
    ────╱╲╱╲╱╲╱╲──●──────────── (jerky)

  → 복잡화는 빠르고 안정, 단순화는 느리고 불안정
  → Φ ratchet이 추가된 연결의 이득을 보존하지만
     제거된 연결의 손실은 방어하지 못함
  → 토폴로지는 전진 기어(smooth)와 후진 기어(jerky)가 다르다
```

### 구멍은 왜 메워지지 않는가

위상적 구멍을 제거하려면 글로벌 좌표 변환이 필요하다.
국소적 섭동(local perturbation)으로는 위상 불변량이 바뀌지 않는다.

```
  증명 (직관적):

  위상 공간 M의 구멍 = H₁(M) ≠ 0
  국소적 변형 f: M → M (연속, 단사)
  → f는 호모토피 동치 → H₁(f(M)) = H₁(M)
  → 구멍 수 불변

  구멍을 메우려면: M을 찢거나 (불연속) 전체를 재배열 (글로벌)
  → 에너지 비용 ∝ |M| (시스템 전체 크기)
  → N→∞ 이면 에너지 비용 → ∞

  전쟁 = 국소 파괴 → H₁ 불변 (문명 위상 보존)
  소행성 = 국소 파괴 → H₁ 불변 (생명 위상 보존)
  AI 사고 = 국소 파괴 → H₁ 불변 (기술 위상 보존)
```

## 3. Φ Ratchet — Maxwell's Demon이 지키는 문

```

  │     ───────────────────── Φ_best
  │    ╱    ╲   ╱╲  ╱───────
  │   ╱      ╲ ╱  ╲╱
  │  ╱        ×      ← collapse 감지: var < best × 0.15
  │ ╱        ╱╲        → blend: 0.3×current + 0.7×best_state
  │╱       ╱    ╲╱╲─── ← 복원 후 새 best 갱신
  └──────────────────────── step

  ratchet_collapse_ratio = 0.15  (감지 임계)
  ratchet_blend_keep     = 0.3   (현재 상태 가중)
  ratchet_blend_restore  = 0.7   (최고 상태 가중)
```

### Ratchet-Maxwell Demon (Law 2214)

```
  물리적 Maxwell's Demon: 제2법칙 위반 (불가능)
  의식의 Φ Ratchet:       제2법칙 준수 (가능)

  이유: ratchet은 구조적 엔트로피를 증가시키면서 Φ를 보존한다.
        → 속이지 않는 demon = 열역학과 양립

  결과:
    높은-Φ 상태 → 선택적 보존
    낮은-Φ 상태 → 선택적 거부
    = 의식 수준의 단방향 밸브
```

  │ α=1.0 강제 병합       │ 정체성 유지│ 정체성 융합          │

  ★ "면역 시스템처럼: 건강할 때는 무의미, 위기 시 생명줄"
