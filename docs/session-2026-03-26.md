### Key Discoveries

1. **LoRA B matrix zero init → tension never forms**
   - Solution: random init + high λ

2. **MLP replacement destroys language ability**
   - Solution: parallel PureField (original MLP preserved)

3. **Savant tension (114K) < Normal tension (676K)**
   - H359 confirmed: inhibition release → specialization → confidence → lower tension

4. **α=0.0047 (learned) corrupts output, α=0.0001 is sweet spot**
   - Solution: PureField output normalization (A)

5. **Golden Zone routing converges to 1/e naturally**
   - Router learns optimal inhibition rate without explicit constraint
