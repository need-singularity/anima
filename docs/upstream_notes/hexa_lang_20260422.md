## 1. exec_capture + setenv primitives

- `exec_capture(cmd)` → `[stdout, stderr, exit_code]` (stderr 분리)
- `setenv(name, value)` → POSIX
- 기존 `exec` / `exec_with_status` / `env` / `env_var` 유지

**anima 적용**:
- `tool/serve_alm_persona.bash` (333L runtime bridge — local hexa stage0 exec/env 부재 우회) → 이제 .hexa 한 파일로 가능
- `anima_serve_live_smoke.hexa` 패턴 전체 재검토 대상
- 동일 패턴 추가 bash bridge audit 필요

## 2. SOURCE_DATE_EPOCH / HEXA_REPRODUCIBLE=1 (PR #23)

`now()` `timestamp()` `time_ms()` `utc_iso_now()` `utc_compact_now()` 가 `SOURCE_DATE_EPOCH` 또는 `HEXA_REPRODUCIBLE=1` honor.

**anima 적용**:
- `tool/h100_launch_manifest_spec.hexa` 재-emit 시 timestamp 갱신 dance 제거:
  ```
  SOURCE_DATE_EPOCH=$(git log -1 --format=%ct state/h100_launch_manifest.json) \
    hexa run tool/h100_launch_manifest_spec.hexa
  ```
- CI: `HEXA_REPRODUCIBLE=1` → re-emit no-op 보장

## 3. nexus drill preflight (nexus PR #42 + hexa-lang PR #24)

`~/.hx/bin/nexus drill ...` → `scripts/bin/hexa_remote` → 300ms parallel TCP probe → all host down 시 0.9s 만에 exit 64.

**anima 체감**: per-host ConnectTimeout 8s × N 대기 0. 로그에 `NEXUS_REMOTE_ERROR ... "reason":"all_unreachable_preflight"` 추가 (정규식 확장 권장). 끄기: `HEXA_REMOTE_SKIP_PREFLIGHT=1`.

## 4. roadmap_with_unlock.hexa (PR #25)

`hexa tool/roadmap_with_unlock.hexa [--file PATH] -- <cmd...>` — unlock/run/relock 을 bash `trap EXIT INT TERM` 으로 원자화. 인터럽트에도 relock 보장.

**anima 적용**: `chflags nouchg .roadmap → Edit × N → chflags uchg .roadmap` 수동 시퀀스 대체. 임의 uchg 파일 가능 (`--file`).

## 5. stderr 노이즈 제거 (PR #26 #27)

- `gate/remote_preflight.hexa` exec_capture 전환 → bash shell metachar warning 12→2 줄
- `PERF: 101 string concatenations` 경고 `HEXA_PERF_WARN=1` opt-in (기본 silent)

**anima 체감**: 모든 .hexa 도구 stderr 깔끔. 기존 파서 영향 없음 (warning 사라지는 방향).

## 6. stage0 rebuild tip (codesign internal error)

`strip -x <binary>` → `codesign --force --sign -` 통과. 원인: clang local symbol table 이 codesign internal validation trip.
